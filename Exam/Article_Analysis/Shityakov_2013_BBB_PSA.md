# Analyzing Molecular Polar Surface Descriptors to Predict Blood-Brain Barrier Permeation

> [!abstract] Краткое резюме
> Работа сравнивает три типа дескрипторов полярной поверхности (2D-PSA, TPSA, 3D-PSA) для предсказания проницаемости гематоэнцефалического барьера (ГЭБ) на наборе 19 CNS-активных/неактивных препаратов. 3D-PSA обеспечивает наилучшую корреляцию с экспериментальными значениями logBB (R² = 0.92, RMSD = 0.29), что демонстрирует эффективность полярных дескрипторов для ранней фармакокинетической оценки. Роль P-гликопротеина как эффлюксного транспортёра ограничивает применимость пассивно-диффузионных моделей.

---

## 🎯 Цель и гипотеза

**Цель:** Оценить предсказательную силу различных дескрипторов полярной поверхности для предсказания скорости проницаемости ГЭБ химическими соединениями. Найти сильную корреляцию между вычисленными дескрипторами и экспериментальными значениями logBB с помощью метода линейных наименьших квадратов с частичным отбором (PLS).

**Гипотеза:** Препараты преимущественно проникают через ГЭБ путём **пассивной диффузии**, на которую сильно влияет общая молекулярная полярность. Если молекула слишком полярна — липидная мембрана не может «вытащить» её из водного растворителя; если недостаточно полярна — она не может вернуться в водную среду мозга. Следовательно, дескрипторы полярной поверхности (PS) — ключевые метрики для предсказания судьбы препарата.

---

## 🧪 Методология

**Набор данных:**
- 19 разнообразных препаратов (CNS-активные и неактивные)
- Экспериментальные значения logBB в стационарном состоянии (из крысиных моделей in vivo, Kelder et al., 1999)
- logBB = log(C_мозг / C_кровь)
  - logBB > 0.5 → хорошо проникает через ГЭБ
  - 0.0–0.5 → умеренная проницаемость
  - −0.3 до 0.0 → слабая проницаемость
  - < −0.3 → практически не проникает

**Вычисленные дескрипторы:**

| Дескриптор | Описание | Программа |
|---|---|---|
| **2D-PSA** | 2D площадь полярной поверхности (атомы O, N, H при полярных атомах) | Spartan'10 |
| **TPSA** | Топологическая PSA (из таблиц вкладов фрагментов, без 3D-координат) | MOE2009 |
| **3D-PSA (PA)** | 3D площадь полярной поверхности через квантово-механические расчёты и карты электростатического потенциала | Spartan'10 + DFT/6-31G* |
| **W2** | Гидрофильный объём (молекулярные поля) | Volsurf+ |

**Вычислительный протокол:**
1. Построение и минимизация 3D-структур молекул (Spartan'10, силовое поле MMFF)
2. DFT-расчёты (метод функционала плотности, базисный набор 6-31G*) для построения карт электростатического потенциала
3. Расчёт 3D-PSA из карт (порог: абсолютное значение электростатического потенциала > 100 кДж/моль)
4. PLS-подгонка дескрипторов к экспериментальным logBB
5. Оценка качества: R² (коэффициент детерминации) и RMSD (корень из среднеквадратичного отклонения)

## 📊 Ключевые результаты

### Таблица результатов корреляции

| Дескриптор | R² | RMSD | p-value |
|---|---|---|---|
| **3D-PSA (PA)** | **0.92** | **0.29** | **< 0.0001** |
| TPSA | 0.85 | 0.39 | < 0.0001 |
| 2D-PSA | 0.71 | 0.54 | < 0.0001 |
| W2 (гидрофильный объём) | 0.42 | 0.77 | 0.0026 |

### Пороговые значения для проницаемости ГЭБ

- Молекулы с PSA **< 60 Å²** обычно проникают через ГЭБ (порог на рис. 2A)
- Молекулы с PSA **> 120–140 Å²** плохо проникают через клеточные мембраны
- W2 (гидрофильный объём) **> 750 Å³** → маловероятно проникновение через ГЭБ

### Выбросы и роль P-гликопротеина

Препараты **icotidine, temelastine, ranitidine, domperidone, 9-hydroxyrisperidone, risperidone** — выбросы, потому что являются субстратами **P-гликопротеина (P-gp)** — эффлюксного транспортёра, который «выкачивает» ксенобиотики из мозга. Эти молекулы имеют «подходящую» PSA, но всё равно не проникают через ГЭБ из-за активного транспорта.

> [!warning] Важное ограничение
> Все модели на основе PSA предполагают, что препараты проникают через ГЭБ исключительно путём **пассивной диффузии**. При наличии активных транспортёров (P-gp, MRP) модель даёт ошибочные предсказания.

## 💡 Выводы авторов

1. Дескрипторы полярной поверхности — **эффективные инструменты** для предсказания фармакокинетических свойств на ранних этапах разработки препаратов.
2. **3D-PSA** обеспечивает наибольшую точность (R² = 0.92), но требует квантово-механических расчётов → не подходит для скрининга огромных баз данных.
3. **TPSA** — хороший баланс точности (R² = 0.85) и скорости расчёта (не нужны 3D-координаты) → предпочтительна для высокопроизводительного скрининга.
4. Необходимо всегда учитывать роль **эффлюксных транспортёров (P-gp)** — они могут существенно ограничивать доступ в мозг даже при «оптимальной» PSA.
5. Проницаемость ГЭБ можно улучшить, снизив общую водородносвязывающую способность молекулы: через **внутримолекулярные H-связи** или защиту полярных групп **неполярными заместителями** (пролекарства).

## 🔗 Связь с курсом хемоинформатики

- **Молекулярные дескрипторы:** прямой пример того, как молекулы представляются как числовые векторы в химическом пространстве (2D vs. 3D дескрипторы)
- **Воронка виртуального скрининга:** TPSA и 2D-PSA — типичные быстрые фильтры первого этапа
- **От данных к знаниям (пирамида DIKW):** экспериментальные logBB → PLS-модель → обобщённые правила проницаемости
- **ГЭБ в фармакологии:** критический параметр ADME, одна из центральных тем курса
- **Пассивная диффузия vs. активный транспорт:** объясняет, почему чисто дескрипторные модели имеют ограничения
