# Folding–unfolding asymmetry and a RetroFold computational algorithm

> [!abstract] Краткое резюме
> Авторы рассматривают фолдинг белка как молекулярную самосборку, а разфолдинг — как дизассемблирование. Поскольку разфолдинг (~5 нс) в два порядка быстрее фолдинга (~800 нс), авторы предлагают алгоритм **RetroFold**: симулировать разфолдинг, применить математическое логарифмическое преобразование времени и получить приближённую траекторию фолдинга с ошибкой <4.1%. Метод тестирован на мини-белке Trp-cage методами МД и МК.

---

## 🎯 Цель и гипотеза

**Проблема:** Прямое моделирование фолдинга белка методом молекулярной динамики (МД) требует колоссальных вычислительных ресурсов — микросекунды и миллисекунды для достижения нативной структуры.

**Гипотеза:** Процессы самосборки (фолдинг) и дизассемблирования (разфолдинг) **асимметричны**, но эту асимметрию можно частично компенсировать математическим преобразованием переменных. Тогда разфолдинг можно рассматривать как **временно-обращённый фолдинг** — аналогично ретросинтезу в органической химии.

**Ключевое утверждение:**
- Фолдинг — экспоненциально затухающий процесс (диссипативные силы доминируют)
- Разфолдинг — процесс с постоянной скоростью (движущие силы уравновешены демпфированием)
- Нет точной симметрии между двумя процессами, но математическое преобразование даёт разумное приближение

## 🧪 Методология

### Объект исследования
**Trp-cage** — наименьший известный белок (20 аминокислотных остатков, полипептид):
- Последовательность: NLYIQWLKDGGPSSGRPPPS
- Характерна гидрофобным захоронением Trp в ядре
- Широко используется как модельная система для вычислительных исследований
- PDB ID: 1L2Y (38 NMR-конформаций)
- Tm = 42°C (в норме); мутации в α-спиральной части повышают до 64°C

### Вычислительные инструменты
| Инструмент | Назначение |
|-----------|-----------|
| **AMBER** (модуль LEaP) | Построение линейной конформации белка |
| **Avogadro** | Создание 3D молекулярной структуры |
| **SANDER** (AMBER) | МД-симуляции (Linux/Unix) |
| **Thermostat Berendsen** | Контроль температуры в МД |
| **SHAKE алгоритм** | Ограничение длины ковалентных связей |
| **ff99 force field** | Силовое поле для МД |
| **Generalized Born model** | Учёт эффектов сольватации |
| **Rosetta** | Рефайнмент конформаций, оценка стабильности (REU) |
| **AlphaFold2** | Сравнительное предсказание (RMSD 0.56 Å) |
| **Monte Carlo (MC)** | Оценка энергетических уровней конформаций |

### Протокол МД-симуляций

**Фолдинг:**
- Старт: вытянутая (линейная) конформация
- Минимизация (500 циклов) → нагрев (50 пс) → равновесие при **325 K, 800 нс**

**Разфолдинг (денатурация):**
- Старт: нативная глобулярная конформация
- МД при **473 K (498 K), 5.0 нс** (высокая температура ускоряет денатурацию)

### Математическое преобразование RetroFold
1. Нормализация потенциальной энергии: начальное E = −707.6 ккал/моль, конечное E = −350 ккал/моль
2. Введение безразмерного времени τ = t/Δt
3. Логарифмическое преобразование: **q = ln τ / (−ζω₀Δt)**, где ζ — коэффициент демпфирования
4. Применение обратного временного перехода: t = q·Δt
5. Аппроксимация фолдинговой кривой экспоненциальной функцией:

**E_pot = −562 кКал/моль + 75 кКал/моль · e^(−t/Δt)**

## 📊 Ключевые результаты

### Асимметрия фолдинга и разфолдинга
| Параметр | Фолдинг | Разфолдинг |
|---------|---------|-----------|
| **Время симуляции** | 800 нс | 5.0 нс |
| **Соотношение** | в **160 раз** дольше | быстрее |
| **Характер процесса** | Экспоненциальное затухание | Постоянная скорость |
| **Потенциальная энергия** | −487 → −562 ккал/моль | −707.6 → −350 ккал/моль |
| **Скорость выброса энергии** | — | 16.25 ккал/моль·пс⁻¹ |

### Точность RetroFold
- Максимальная ошибка: **4.1%** при t = 524 пс
- Для t > 1 нс: ошибка **< 3.5%**
- Коэффициент аппроксимации: **r² = 0.99**

### Стабильность NMR-конформаций (MC-анализ)
- Свёрнутая форма Trp-cage: энергия от **−28 до −22 REU**
- Развёрнутая форма: энергия от **−22 до −10 REU**
- Минимальные значения: −28.45 REU (свёрнутая) vs −22.0 REU (развёрнутая)
- RMSD порог: < 1.0 Å (подтверждён трёхмерным выравниванием)

### AlphaFold2 для сравнения
- RMSD = 0.56 Å (близко к идеальному 0.5 Å)
- Полный фолдинг занял бы ~4 мкс — RetroFold даёт приближение за значительно меньшее время

## 💡 Выводы авторов

1. **Асимметрия подтверждена количественно**: разфолдинг в >100 раз быстрее фолдинга, оба процесса качественно различны
2. **RetroFold работает**: симуляция разфолдинга (~5 нс) + математическое преобразование даёт приближение с ошибкой <4.1%
3. **Вычислительная экономия**: требуется на два порядка меньше ресурсов по сравнению с прямой симуляцией фолдинга
4. **Ограничение**: алгоритм пропускает детали (колебания между метастабильными состояниями), но даёт общую картину траектории самосборки
5. **Связь с неэргодичностью**: асимметрия фолдинга связана с неэргодической природой процесса; математическое преобразование аналогично преобразованию Ламперти

## 🔗 Связь с курсом хемоинформатики

- **Молекулярная динамика**: прямое применение МД для изучения фолдинга/разфолдинга (AMBER, ff99, Berendsen, SHAKE)
- **Алгоритмы фолдинга**: RetroFold — новый вычислительный алгоритм, дополняющий AlphaFold2, RoseTTAFold, mfold
- **Monte Carlo**: использован для оценки энергетики конформаций
- **Аналогия с ретросинтезом**: RetroFold назван по аналогии с ретросинтетическим анализом в органической химии (обратное планирование пути)
- **RMSD**: ключевой параметр МД для оценки стабильности и качества структуры
- **Нотация PDB**: работа с PDB-структурами (1L2Y), NMR-конформациями
- **Синтетическая биология**: исследование фолдинга как самосборки молекулярных систем

## ❓ Возможные вопросы на экзамене

1. **Что такое RetroFold и почему этот алгоритм получил такое название?**
   → По аналогии с ретросинтезом: алгоритм «обращает» процесс разфолдинга

2. **В чём принципиальная асимметрия фолдинга и разфолдинга?**
   → Фолдинг — экспоненциально медленный (диссипация); разфолдинг — быстрый, постоянная скорость

3. **Какой белок использован как модельная система и почему?**
   → Trp-cage, 20 АК, наименьший белок, хорошо изучен, удобен для вычислений

4. **Какова точность RetroFold и как её измеряли?**
   → Ошибка <4.1% (максимум при t=524 пс), r²=0.99

5. **Как связаны фолдинг белка и концепция неэргодичности?**
   → Неэргодичные системы не достигают всех микросостояний с равной вероятностью; фолдинг — путь к единственному нативному состоянию; преобразование Ламперти компенсирует неэргодичность
