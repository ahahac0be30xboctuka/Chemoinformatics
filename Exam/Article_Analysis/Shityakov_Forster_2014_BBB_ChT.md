# In silico predictive model to determine vector-mediated transport properties for the blood–brain barrier choline transporter

> [!abstract] Краткое резюме
> Авторы разработали in silico структурно-базированную модель для скрининга лекарственных молекул, способных использовать холиновый транспортёр ГЭБ (BBB-ChT) в качестве вектора доставки в ЦНС. Молекулярный докинг (AutoDock и AutoDock Vina) на гомологичной модели ChT1 показал высокую корреляцию предсказанных и экспериментальных значений (r²=0.88), AUC=0.82/0.81 для двух программ на датасете из 93 соединений.

---

## 🎯 Цель и гипотеза

**Проблема:** ≈98% лекарственных веществ не проникают через ГЭБ. Пассивная диффузия ограничена физико-химическими свойствами молекулы (LogP, PSA). Холиновый транспортёр ГЭБ (BBB-ChT) — потенциальный вектор активной доставки катионных молекул в ЦНС, поскольку in vivo он занят лишь на ~25% от Km при нормальной концентрации холина в плазме.

**Ограничение:** BBB-ChT не клонирован и не кристаллизован → нет 3D-структуры → невозможно проводить прямой докинг.

**Гипотеза:** Гомологичная модель ChT1 (построенная по кристаллической структуре натрий-галактозного симпортера, PDB: 3DH4) может служить суррогатом BBB-ChT для виртуального скрининга молекул-кандидатов для доставки через ГЭБ.

## 🧪 Методология

### Построение модели белка
- **Шаблон для гомологичного моделирования:** кристаллическая структура натрий/галактозного симпортера (PDB ID: 3DH4) — предоставлена профессором Ябуучи (Киотский университет)
- **Электростатические карты потенциала:** программа Delphi v5.1 (решение уравнения Пуассона–Больцмана конечными разностями)
- **Визуализация:** UCSF Chimera v1.7

### Датасет соединений
- **93 соединения** из литературы: 44 активных (BBB-ChT-активных) и 49 неактивных
- Включает бис-пиридиниевые циклофаны, аналоги аммония, донепезил (референс)

### Молекулярный докинг
- **AutoDock** и **AutoDock Vina (ADVina)** — два алгоритма для взаимной валидации
- **PyRx** — оптимизация датасета и управление докингом
- Белок считался жёстким, лиганды — гибкими
- Параметр оценки: **ΔGbind** (свободная энергия связывания, ккал/моль)
- Диапазон: −10.49 до −2.66 ккал/моль (AutoDock); −11.117 до −2.0 ккал/моль (ADVina)
- **Порог разделения:** ΔGbind < −6.0 ккал/моль → активное; > −6.0 → неактивное

### Физико-химические дескрипторы (VolSurf+)
- **ClogP** — расчётный коэффициент октанол/вода (липофильность): −5.87 до 5.204
- **PSA** — полярная поверхность: 3.24–64.67 Å²
- **logBBpassive** — логарифм соотношения концентраций мозг/плазма при пассивной диффузии:
  `logBBpassive = 0.152·ClogP − 0.0148·PSA + 0.139` (уравнение Кларка)
- **logBBactive** — с учётом активного транспорта через BBB-ChT:
  `logBBactive = logBBpassive · pKiexp`

### Статистический анализ
- Линейная регрессия (GraphPad Prism v4)
- ROC-кривые (кривые ошибок), AUC
- MATLAB R2012a для линейных зависимостей logBB

## 📊 Ключевые результаты

### Предсказательная сила модели
| Параметр | Значение |
|----------|---------|
| r² (ΔGbind predicted vs experimental) | **0.88** |
| F-статистика | 692.4 |
| Стандартная ошибка оценки (SEE) | 0.775 |
| P-value | < 0.0001 |
| AUC (AutoDock) | **0.82** |
| AUC (ADVina) | **0.81** |

### Кластеризация активных/неактивных
- Обе программы успешно кластеризовали активные (ΔGbind < −6.0) и неактивные (ΔGbind > −6.0) соединения
- **Ложноположительные (AutoDock/ADVina):** 3 соединения, ΔGbind от −8.68 до −6.49 ккал/моль
- **Ложноотрицательные:** 5 соединений, ΔGbind от −4.32 до −3.06 ккал/моль

### Смещение по размеру молекулы (size-related bias) ⚠️
- Соединения с **< 20 тяжёлых атомов** → ложноотрицательные результаты
- Соединения с **≥ 20 тяжёлых атомов** → ложноположительные результаты
- ROC-анализ по количеству тяжёлых атомов: AUC оценки = 0.814 → размер молекулы является конфаундером

### Корреляция pKi (предсказанный vs экспериментальный)
- r² = 0.47 (умеренная) — авторы отмечают, что для 90 комплексов из PDB статистически значимой считается корреляция при r² = 0.45–0.55

### Физико-химические свойства ключевых соединений
| Соединение | Kiexp (μM) | ΔGbind AutoDock | ΔGbind ADVina | ClogP | PSA (Å²) | logBBpassive |
|-----------|-----------|-----------------|---------------|-------|-----------|-------------|
| 5d | 33.8 | −10.2 | −11.11 | 1.16 | 8.66 | 0.187 |
| 5c | 0.8 | −9.92 | −10.74 | 0.74 | 8.66 | 0.123 |
| 5b | 1.4 | −8.71 | −10.04 | 0.313 | 8.66 | 0.058 |
| Донепезил | 274* | — | — | 3.08–4.11 | 38.8 | 0.89 |

### Эффект активного транспорта
- Бис-пиридиниевые циклофаны (5b, 5c, 5d) при **пассивной диффузии** демонстрируют низкое проникновение в мозг
- При учёте **активного транспорта через BBB-ChT** — концентрация в мозге значительно возрастает
- → BBB-ChT усиливает доставку катионных лекарств в ЦНС

### Механизм связывания
- Активные соединения занимают **отрицательно заряженные участки поры** транспортёра (ближе к центру трансмембранного канала)
- Ключевое взаимодействие: **ионное** между триметиламмониевым фрагментом лиганда и аминокислотными остатками белка
- Метильные акцепторы полости потенцируют аффинность к BBB-ChT

## 💡 Выводы авторов

1. In silico структурно-базированная модель на основе гомологии ChT1 является **высокопредсказательным инструментом** для идентификации молекул, использующих BBB-ChT для проникновения в мозг (r²=0.88, AUC=0.82)
2. До получения кристаллической структуры человеческого BBB-ChT этот гомологичный подход даёт эффективную стратегию **рационального дизайна ЦНС-направленных фармацевтиков**
3. Обнаружено **смещение докинга по размеру молекулы** (< 20 тяжёлых атомов → ложноотрицательные результаты) — ключевой методологический вывод
4. Активный транспорт через BBB-ChT **значимо улучшает** доставку в ЦНС по сравнению с пассивной диффузией
5. Модель позволяет снизить зависимость от дорогостоящих конвенциональных QSAR-методов

## 🔗 Связь с курсом хемоинформатики

- **Молекулярный докинг:** применение двух алгоритмов (AutoDock / ADVina) → структурно-базированный виртуальный скрининг
- **ГЭБ и BBB-параметры:** logBBpassive, logBBactive, ClogP, PSA — ключевые дескрипторы проникновения через ГЭБ (Лекции 1, 8)
- **Гомологичное моделирование:** замена недоступной структуры мишени → типичная хемоинформатическая задача при отсутствии кристаллической структуры
- **Метрики оценки модели:** ROC-AUC, r², SEE, F-статистика — стандартные метрики Python/sklearn
- **Физические смещения докинга:** size-related bias → обсуждается в лекции о молекулярном докинге (стандартная ошибка для крупного кластера)
- **Холиновый транспортёр как вектор:** пример активного транспорта при разработке ЦНС-препаратов

## ❓ Возможные вопросы на экзамене

1. **Почему использовали гомологичную модель ChT1 вместо прямой структуры BBB-ChT?** → BBB-ChT не клонирован и не кристаллизован; ChT1 (PDB: 3DH4) — ближайший структурный аналог
2. **Что такое size-related bias в докинге, и почему это проблема?** → Молекулы с < 20 тяжёлых атомов получают ложноотрицательные оценки; причина — меньший конформационный поиск, меньше торсионных связей → алгоритм недооценивает связывание
3. **Чем отличается logBBpassive от logBBactive?** → passive: только диффузия по уравнению Кларка (ClogP, PSA); active: с коррекцией на pKiexp транспортёра
4. **Какие метрики использовались для оценки модели?** → r²=0.88 (ΔGbind), AUC=0.82/0.81, F=692.4, SEE=0.775, p<0.0001
5. **Какова практическая ценность этой модели?** → Дешёвый скрининг кандидатов для ЦНС-доставки до синтеза; снижение затрат на QSAR; выявление молекул с потенциалом активного транспорта
