## Вопрос 1: Что такое молекулярный дескриптор? Примеры. Какие дескрипторы используются в исследованиях ГЭБ?

> [!tip] Ключевые тезисы
> - **Дескриптор** — числовое значение, кодирующее свойство молекулы (структура → число)
> - **1D** — из формулы (MW, HBD, HBA); **2D** — из графа (logP, PSA, фингерпринты); **3D** — из геометрии (SASA, объём, VE)
> - Для ГЭБ главные предикторы: **PSA < 90 Å²**, **logP ≤ 5**, **MW < 500 Да**, **HBD ≤ 3**
> - Инструмент расчёта: **RDKit** (Python, open source)
> - Всего существует > 5000 дескрипторов

Молекулярный дескриптор — это число, которое описывает какое-то свойство молекулы. Представьте, что вы хотите научить компьютер работать с химией — для этого нужно перевести структуру молекулы в цифры. Дескрипторы делятся по уровню детализации: **1D-дескрипторы** считаются прямо из формулы (например, **молекулярная масса MW** или число атомов-доноров водородной связи **HBD**). **2D-дескрипторы** строятся из графа молекулы без учёта геометрии — важнейшие из них **logP** (липофильность) и **TPSA** (полярная площадь поверхности). **3D-дескрипторы** требуют пространственной структуры: например, **SASA** (площадь, доступная растворителю) или **Voronoi Entropy**. Для предсказания проникновения через **гематоэнцефалический барьер (ГЭБ)** ключевые дескрипторы: PSA < 90 Å² (предиктор №1), logP ≤ 5, MW < 500 Да, HBD ≤ 3. Лучший ML-классификатор ГЭБ (XGBoost) достигает **AUC = 0.96** с комбинацией RDKit-дескрипторов и фрагментарных фингерпринтов. Расчёт дескрипторов на практике выполняется в библиотеке **RDKit**.

## Вопрос 2: Типичные химические форматы в хемоинформатике и синтетической биологии

> [!tip] Ключевые тезисы
> - **SMILES** — компактная строка для малых молекул (самый популярный)
> - **SDF/MDL** — хранение структур + биоактивность (IC₅₀, Ki); мультиструктурный
> - **PDB** — белки и нуклеиновые кислоты; >200 000 структур
> - **InChI/InChIKey** — уникальная идентификация молекул (стандарт IUPAC)
> - **OpenBabel** — конвертер >100 форматов; проблема: ароматика при конвертации

Каждый тип молекул хранится в «своём» формате. **SMILES** (Simplified Molecular Input Line Entry System) — самый распространённый: молекула записывается как строка символов. Бензол — `c1ccccc1`. Очень компактно, идеально для больших библиотек (ZINC — 35 млн молекул), но не отражает 3D-геометрию. **SDF/MDL** — стандарт для хранения набора молекул вместе с активностью (IC₅₀, Ki). Именно из SDF загружаются данные из **BindingDB**. **PDB** — формат для белков: каждая строка описывает один атом с координатами (XYZ), остатком и цепью; лиганды помечаются как HETATM. Создан в 1971 году, содержит >200 000 структур. **InChI/InChIKey** — стандарт IUPAC, гарантирует уникальность: InChIKey — 27-символьный хеш для быстрого поиска в базах. Для синтетической биологии (гены, праймеры) используется **FASTA**. Конвертацию между форматами выполняет **OpenBabel**; главная ловушка — ароматические кольца (Kekulé vs. ароматическое представление) дают ошибки.

## Вопрос 3: Основные молекулярные дескрипторы в QSAR/QSPR-исследованиях

> [!tip] Ключевые тезисы
> - **QSAR** (Structure-Activity) и **QSPR** (Structure-Property) — модели «структура → число»
> - 2D-дескрипторы — самый популярный класс: **logP, TPSA, MR, csp3, QED**
> - **Правило Липинского (Rule of Five)**: MW ≤ 500, logP ≤ 5, HBD ≤ 5, HBA ≤ 10
> - Фингерпринты (ECFP/Morgan, MACCS) — бинарные векторы подструктур для ML
> - Новый 3D-дескриптор: **Voronoi Entropy** — мера асимметрии молекулы

**QSAR** — это математическая модель: Активность = f(дескрипторы). Смысл прост: если молекулы с похожей структурой имеют похожую активность, то между числами-дескрипторами и активностью должна быть закономерность. Самые важные **2D-дескрипторы**: **logP** (мера жирности — насколько молекула «любит» масло vs. воду), **TPSA** (полярная площадь — чем больше, тем хуже всасывание), **csp3** (доля насыщенных атомов C), **QED** (общая «лекарственность» от 0 до 1). Топологические индексы (**Wiener, Randić, Balaban**) описывают форму графа молекулы. **Молекулярные фингерпринты** (ECFP/Morgan, MACCS, Avalon) — бинарные векторы, фиксирующие наличие подструктур; они стандарт для задач поиска сходства и машинного обучения. **Правило Липинского**: если ≥2 из пяти условий нарушены — молекула, скорее всего, не всасывается перорально. Из **3D-дескрипторов** самый новый — **Voronoi Entropy (VE)**: мера асимметрии атомного расположения, показавшая r² = 0.94 с аффинностью к Mpro SARS-CoV-2.

## 📄 Статья: Shityakov et al., 2023 — Voronoi Entropy as a Ligand Molecular Descriptor

> [!abstract] Карточка статьи
> **Цель:** предложить новый молекулярный дескриптор — **Voronoi Entropy (VE)** — и проверить его корреляцию с аффинностью лигандов к главной протеазе SARS-CoV-2 (**Mpro**).
>
> **Метод:** 1144 лиганда из **BindingDB** (активность IC₅₀). VE рассчитывается через **RDKit**: строится 2D-диаграмма Вороного атомов, вычисляется $S_v = -\sum P_n \ln P_n$, где $P_n$ — доля n-угольных ячеек. Линейная регрессия IC₅₀ ~ VE. Валидация: докинг AutoDock + МД 100 нс (GROMACS) + MM-PBSA/GBSA.
>
> **Главный результат:** **IC₅₀ = −274.9 · S_v + 613 нМ**, r² = **0.94**, p = **0.0003**. Шеннонова энтропия по SMILES-строке — p > 0.05 (не коррелирует).
>
> **Вывод:** Пространственная асимметрия молекулы (а не её последовательность символов) определяет аффинность к рецептору. Чем выше VE → тем асимметричнее молекула → тем активнее связывается.

> [!note] Факты, которые точно спросят
> - **Формула:** $S_v = -\sum P_n \ln P_n$ (аналог энтропии Шеннона, но по геометрии Вороного)
> - **Диапазон VE:** 0.4 ≤ S_v ≤ 1.66 (распределение ≈ Гауссово)
> - **Тест-молекулы:** соед. 1 (S_v = 1.41, IC₅₀ = **15 нМ**) и соед. 2 (S_v = 0.78, IC₅₀ = **667 нМ**)
> - **Докинг:** ΔG = −9.95 vs −9.36 ккал/моль
> - **МД:** RMSD соед. 1 стабилен ~3 Å; соед. 2 флуктуирует до 8 Å
> - Контроль: Shannon Entropy по SMILES **не коррелирует** (p > 0.05) — это ключевой аргумент в пользу геометрической природы дескриптора
