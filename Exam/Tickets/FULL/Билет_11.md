## ❓ Вопрос 1: Что такое молекулярный дескриптор? Примеры. Какие дескрипторы используются в исследованиях ГЭБ?

> [!tip] Ключевые тезисы
> - Молекулярный дескриптор — численное значение, кодирующее свойство молекулы
> - Классификация: 1D (формульные), 2D (топологические), 3D (геометрические)
> - Для ГЭБ наиболее важны: PSA, logP, MW, HBD, HBA, logBB
> - Инструмент расчёта: RDKit (Python), существует >5000 дескрипторов

### Развёрнутый ответ

**Молекулярный дескриптор** — численная характеристика молекулы, получаемая из её структурного представления (графа или вектора). Дескрипторы превращают химические структуры в числа, пригодные для математического анализа, машинного обучения и построения моделей QSAR/QSPR.

#### Классификация дескрипторов по размерности

**1D-дескрипторы** вычисляются непосредственно из молекулярной формулы:
- **MW** (молекулярная масса) — один из важнейших фильтров (правило Липинского: MW ≤ 500 Да)
- **Число атомов** (тяжёлых, ароматических, гетероатомов)
- **Число H-доноров (HBD) и H-акцепторов (HBA)**
- **Число вращаемых связей**

**2D-дескрипторы** получают из топологического графа (без 3D-координат):
- **logP** — коэффициент распределения октанол/вода; мера липофильности; рассчитывается по методу Wildman–Crippen
- **PSA / TPSA** (polar/topological polar surface area) — полярная площадь поверхности; для ГЭБ критично < 90 Å²
- **MR** (молярная рефрактивность) — мера поляризуемости молекулы
- **Топологические индексы**: Wiener, Randić, Balaban — описывают связность графа молекулы
- **Молекулярные фингерпринты**: Morgan/ECFP, MACCS, Avalon, MaxFP — бинарные векторы подструктур
- **csp3** — доля sp3-атомов углерода (насыщенность)
- **QED** (quantitative estimate of drug-likeness) — интегральная оценка лекарственности
- **Правила Липинского**: MW ≤ 500, logP ≤ 5, HBD ≤ 5, HBA ≤ 10

**3D-дескрипторы** требуют 3D-конформации молекулы:
- **Молекулярный объём** — важен для подбора молекулы к размеру сайта связывания
- **SASA** (solvent accessible surface area) — площадь, доступная растворителю
- **H-SASA** — гидрофобная часть SASA
- **Электростатические потенциалы** (заряды атомов)
- **Воронои-энтропия (VE)** — новый дескриптор симметрии/асимметрии, основан на диаграмме Вороного атомных координат

#### Дескрипторы для исследований ГЭБ

Для предсказания проникновения веществ через гематоэнцефалический барьер ключевыми являются:

- **logBB** — логарифм отношения концентраций в мозге и крови; основная целевая переменная при регрессионных QSAR-моделях ГЭБ
- **PSA/TPSA** — предиктор № 1; PSA < 90 Å² ассоциирована с хорошим проникновением; выше 120 Å² — как правило непроницаемо
- **logP** — чем выше, тем лучше проникновение через липидную мембрану (но нельзя превышать ~5)
- **MW** — предпочтительно < 500 Да; крупные молекулы не проникают
- **HBD** ≤ 3 — водородные доноры замедляют проникновение через мембрану
- **H-SASA** — гидрофобная поверхность коррелирует с мембранной проникаемостью
- **Число нарушений правил Липинского** — чем меньше, тем лучше

> В Лекции 8 (МО): при классификации молекул на BBB+ и BBB− лучший результат достигнут методом XGBoost с AUC = 0.96, используя комбинацию **RDKit-дескрипторов + фрагментарных фингерпринтов**.

Для практического расчёта всех дескрипторов используется библиотека **RDKit** (Python, open source, разработан Greg Landrum).

## ❓ Вопрос 2: Типичные химические форматы в хемоинформатике и синтетической биологии?

> [!tip] Ключевые тезисы
> - SMILES — самый распространённый для малых молекул (строка, компактно)
> - SDF/MDL — стандарт хранения 2D/3D структур + активность
> - PDB — для белков, нуклеиновых кислот; > 200 000 структур
> - InChI/InChIKey — уникальная идентификация (IUPAC)
> - OpenBabel — конвертер между форматами (открытый исходный код)

### Развёрнутый ответ

Химические данные хранятся в специализированных форматах, каждый из которых оптимизирован для конкретного применения.

#### Форматы для малых молекул (лиганды, лекарственные соединения)

**SMILES** (Simplified Molecular Input Line Entry System):
- Разработан Daylight Chemical Systems (Weininger)
- Линейная строковая нотация: атомы — буквы, связи — знаки, кольца — числа
- Пример: бензол = `c1ccccc1`, аспирин = `CC(=O)Oc1ccccc1C(=O)O`
- **Плюсы:** компактность, минимальное хранилище, идеален для HTS, читаем
- **Минусы:** неоднозначность (одна молекула — разные SMILES строки), неудобен для неорганики, не отражает 3D-геометрию
- Разновидности: каноническая SMILES, изомерная SMILES (содержит стереохимию), SMARTS (для поиска подструктур)
- Широко используется в: ZINC (35+ млн молекул), ChEMBL, PubChem, AlindaDB

**SDF / MDL Molfile** (Structure-Data File):
- Разработан Molecular Design Limited
- Хранит 2D и 3D структуры + биологическую активность (IC₅₀, Ki и др.)
- Может содержать несколько молекул в одном файле
- **Применение:** загрузка данных из BindingDB (использован в статье Билета 11), скрининг библиотек
- Поддерживается большинством ПО: VMD, Maestro/Schrödinger, Spartan, Avogadro, RDKit

**MOL2** (Tripos format):
- Формат компании Tripos (программа SYBYL)
- Содержит: координаты атомов, типы атомов (sp2, sp3 и др.), частичные заряды
- Широко применяется в молекулярном докинге (AutoDock, Glide)

**InChI / InChIKey** (IUPAC):
- Стандартизированная нотация, гарантирует уникальность идентификатора
- InChIKey — хешированная форма (27 символов) для быстрого поиска в базах

#### Форматы для макромолекул

**PDB** (Protein Data Bank):
- Разработан Брукхейвенской национальной лабораторией (1971)
- Структура файла: заголовок → атомные записи (тип, имя, остаток, цепь, координаты XYZ) → B-факторы → связи → лиганд (HETATM)
- Применяется для: белков, нуклеиновых кислот, олигосахаридов, вирусов
- >200 000 структур: рентгеноструктурный анализ (>130 000), ЯМР (~12 000), крио-ЭМ
- Лиганды в PDB: добавляются в конец файла как HETATM
- Замена: mmCIF (более современный) — но ПО пока слабо адаптировано

**FASTA:**
- Формат для аминокислотных и нуклеотидных последовательностей
- Используется в синтетической биологии при работе с генами, белками, праймерами
- Пример: `>sp|P12345|PROT_HUMAN Protein\nMASGTGGAALSIK...`

#### Другие форматы

| Формат | Применение |
|---|---|
| XYZ | Координаты атомов; простой формат для квантовой химии |
| CAR | Material Studio; DFT и МД расчёты |
| CASTEP | Квантово-механические расчёты кристаллических структур |
| CIF | Кристаллографическая информация (рентгеноструктурный анализ) |
| GRO / TOP | GROMACS (МД-симуляции): координаты + топология |

#### Конвертация форматов

**OpenBabel** — открытый конвертер между >100 форматами:
- Командная строка: `obabel input.smi -O output.sdf`
- Python-интеграция: `import openbabel`
- Проблемная зона: **ароматические кольца** (Kekulé vs. ароматическое представление) — часто ошибки при конвертации

> «Иногда конвертация очень трудна — иногда приходится делать ручную корректировку базы данных» (Лекция 5).

## ❓ Вопрос 3: Основные молекулярные дескрипторы в QSAR/QSPR-исследованиях?

> [!tip] Ключевые тезисы
> - QSAR: Quantitative Structure-Activity Relationship — количественная связь структура–активность
> - QSPR: Quantitative Structure-Property Relationship — структура–свойство
> - Три уровня дескрипторов: 1D (формула), 2D (топология), 3D (геометрия)
> - Правило Липинского (Rule of Five) — набор ключевых дескрипторов для drug-likeness
> - Новый дескриптор: Voronoi Entropy (VE) — мера симметрии/асимметрии молекулы

### Развёрнутый ответ

**QSAR** (Quantitative Structure-Activity Relationship) — математическая модель, связывающая числовые характеристики молекулы (дескрипторы) с её биологической активностью. **QSPR** — аналогичный подход для физико-химических свойств (растворимость, температура кипения и т.д.).

Общая форма модели:
$$\text{Активность} = f(D_1, D_2, \ldots, D_n)$$

где $D_i$ — молекулярные дескрипторы.

#### 1D-дескрипторы (из молекулярной формулы)

- **Молекулярная масса (MW)** — наиболее фундаментальный; правило Липинского: MW ≤ 500 Да
- **Число тяжёлых атомов (HAC)** — коррелирует с MW
- **H-доноры (HBD)** — OH, NH группы; правило: ≤ 5
- **H-акцепторы (HBA)** — O и N атомы; правило: ≤ 10
- **Число вращаемых связей (RotBonds)** — мера гибкости; правило Вебера: ≤ 10
- **Формула состава** — соотношение C, H, N, O, S, галогены

#### 2D-дескрипторы (из топологического графа)

Это наиболее широко используемый класс в QSAR:

- **logP** — главный дескриптор липофильности; определяет распределение между фазами октанол/вода; расчёт: метод Wildman–Crippen (в RDKit `Descriptors.MolLogP`)
- **PSA / TPSA** (topological polar surface area) — полярная площадь поверхности; рассчитывается как сумма площадей O, N, S атомов; предиктор всасывания и проникновения через ГЭБ
- **MR** (молярная рефрактивность) — мера объёма и поляризуемости; правило Липинского: MR ≤ 130 см³/моль
- **csp3** — доля sp3-атомов C; высокое csp3 = насыщенность = лучшая растворимость
- **QED** — интегральная оценка «лекарственности» (0–1)
- **Топологические индексы:**
  - **Wiener index** — сумма кратчайших расстояний между атомами
  - **Randić index** — корреляция с логарифмом растворимости
  - **Balaban J-index** — характеристика молекулярной разветвлённости
- **Молекулярные фингерпринты** (бинарные векторы подструктур):
  - **Morgan/ECFP** (Extended Connectivity Fingerprints) — кругово обходящие фингерпринты, стандарт в similarity search
  - **MACCS keys** — 166 ключей, описывающих наличие/отсутствие структурных мотивов
  - **Avalon** — широко используется в сходстве (minkowski similarity)
  - **MaxFP, Cats2D, FCFP6** — различные реализации для ML-моделей
  - **Фрагментарные фингерпринты** — на основе специфических химических фрагментов

#### 3D-дескрипторы (из 3D-конформации)

- **Молекулярный объём** — критически важен для подбора к размеру сайта связывания
- **SASA** (solvent accessible surface area) — площадь поверхности, доступная растворителю
- **H-SASA** — гидрофобная часть SASA; коррелирует с мембранной проникаемостью
- **Заряды атомов** — частичные заряды ESP (electrostatic potential), RESP
- **Voronoi Entropy (VE)** — мера асимметрии/упорядоченности молекулы по диаграмме Вороного; при более высокой VE лиганды проявляют большую подвижность и лучше связываются с рецепторами (r² = 0.94, p = 0.0003 на примере Mpro SARS-CoV-2)

#### Правила Липинского (Правило пяти — Rule of Five)

Для предсказания перорального всасывания лекарственного кандидата:

| Дескриптор | Пороговое значение |
|---|---|
| MW | ≤ 500 Да |
| logP | ≤ 5 |
| HBD (H-доноры) | ≤ 5 |
| HBA (H-акцепторы) | ≤ 10 |
| MR | ≤ 130 |

При ≥ 2 нарушениях молекула, скорее всего, не является оральнобиодоступной.

#### Пример применения QSAR

В курсе демонстрировался пример с **корозионными ингибиторами** (бензотриазолы): единственный дескриптор logP позволил построить линейную QSAR-модель, предсказывающую активность. HOMO-LUMO gap дополнительно коррелировал с lipophilicity и активностью.

> «Правило: если это ГЭБ-исследование — используй PSA и logP. Если токсичность — тяжёлые атомы и металлы. Если аффинность — торсионные углы и насыщенность» (Лекция 1).

## 📄 Статья: Voronoi Entropy as a Ligand Molecular Descriptor

> [!summary] Краткий обзор для экзамена
> Шитяков и соавторы (2023) предложили **энтропию Вороного (VE)** в качестве нового молекулярного дескриптора для предсказания аффинности лиганд–белок. VE рассчитывается из SMILES через RDKit: строится 2D-граф молекулы, затем диаграмма Вороного, и вычисляется $S_v = -\sum P_n \ln P_n$. На выборке 1144 лигандов к главной протеазе SARS-CoV-2 (BindingDB, IC₅₀) выявлена сильная линейная корреляция: **IC₅₀ = −274.9·S_v + 613 нМ** (r² = 0.94, p = 0.0003). В отличие от VE, Шеннонова энтропия по SMILES-строке не показала значимой корреляции (p > 0.05), что доказывает: пространственное расположение атомов, а не последовательность символов, определяет аффинность.

> [!note] Подробный анализ
> Полный разбор статьи: [[Shityakov_2023_Voronoi_Entropy]]

### Ключевые факты для экзамена

- **Дескриптор:** Voronoi Entropy (VE) — $S_v = -\sum_n P_n \ln P_n$, где $P_n$ — доля n-сторонних многоугольников Вороного
- **Данные:** 1144 лиганда из BindingDB (COVID-19), активность — IC₅₀ к Mpro SARS-CoV-2
- **Главный результат:** IC₅₀ = −274.9·S_v + 613 нМ, **r² = 0.94**, p = 0.0003
- **Диапазон VE:** 0.4 ≤ S_v ≤ 1.66 (распределение близко к Гауссову)
- **Контрольный эксперимент:** SE (Шеннонова по SMILES) — p > 0.05, корреляции нет
- **Тест-молекулы:** соединение 1 (S_v = 1.41, IC₅₀ = 15 нМ) >> соединение 2 (S_v = 0.78, IC₅₀ = 667 нМ)
- **Докинг (AutoDock):** ΔG = −9.95 ккал/моль (соед. 1) vs −9.36 (соед. 2)
- **МД 100 нс:** RMSD стабилен ~3 Å; соединение 2 флуктуирует до 8 Å (высокая гибкость из-за симметрии)
- **Инструменты:** RDKit (Python), AutoDock, GROMACS, MM-PBSA/GBSA
- **Вывод:** молекулярная подвижность (асимметрия) определяет аффинность — пространственное расположение, а не SMILES-строка
