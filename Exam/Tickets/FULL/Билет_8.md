## ❓ Вопрос 1: Программное обеспечение для молекулярной динамики и примеры применения

> [!tip] Ключевые тезисы
> - Основные пакеты: **GROMACS**, **AMBER**, **NAMD**, **Desmond**
> - Все современные пакеты поддерживают GPU-ускорение
> - Нобелевская премия по химии 1999 г. — за разработку МД-методов
> - Выбор ПО зависит от задачи, бюджета и доступного оборудования

### Развёрнутый ответ

Молекулярная динамика — численный метод исследования многочастичных систем путём численного решения уравнений движения Ньютона (F = ma) для каждого атома системы. Это детерминированный метод: состояние системы в любой момент будущего определяется начальными условиями.

**GROMACS (GROningen MAchine for Chemical Simulations)**
Один из наиболее широко используемых МД-пакетов в академической среде. Поддерживает силовые поля AMBER, CHARMM, GROMOS, OPLS. Открытый исходный код, бесплатен. Масштабируется на CPU и GPU. Богатый набор инструментов анализа траекторий (gmx rms, gmx rmsf, gmx gyrate, gmx hbond). Применяется для белок-лигандных комплексов, мембранных белков, нуклеиновых кислот.

**AMBER (Assisted Model Building with Energy Refinement)**
По мнению преподавателя — лучшее ПО для белок-лигандных МД-симуляций. Разработан в Калифорнийском университете в 2002 г. Включает **AmberTools** (полностью бесплатный с 2022 г.) и AMBER (коммерческий). GPU-модуль **pmemd.cuda** — один из самых быстрых доступных движков. Инструмент анализа — **CPPTRAJ** (расчёт RMSD, RMSF, Rg, водородных связей, PCA, Марковских моделей). Имеет скрипты MM-PBSA/MM-GBSA для расчёта свободной энергии связывания. Совместим с CHARMM-GUI для подготовки систем.

*Пример применения:* симуляция пептид-циклодекстриновых комплексов для оценки стабильности комплекса, водородных связей и свободной энергии; симуляция мембранных рецепторов (погружение в липидную бислой).

**NAMD (Nanoscale Molecular Dynamics)**
Разработан в Университете Иллинойса в 1995 году. Масштабируем (CPU/GPU). Визуальный интерфейс — **VMD** (Visual Molecular Dynamics), который также используется для анализа траекторий из GROMACS и AMBER. Применяется в нанобиотехнологии: симуляция нанопор для секвенирования ДНК, нанотрубок.

**Desmond (Schrödinger)**
Разработан в Schrödinger Inc. Ускоренная GPU-версия в **60–80 раз быстрее** CPU-версии. Интегрирован с Maestro GUI (удобная система подготовки симуляций). Поддерживает расчёты свободной энергии (FEP — free energy perturbation), метод репличного обмена. Совместим с CHARMM, AMBER, OPLS.

**QM/MM гибридный подход**
Комбинация квантовой механики (DFT) для активного центра и классической МД для остального белка. Позволяет изучать механизмы химических реакций (перенос заряда, разрыв связей). Требует высокопроизводительных вычислительных кластеров.

**Историческое значение:** Нобелевская премия по химии 1999 года — Мартин Карплюс, Майкл Левитт и Ариех Варшел. МД сегодня рассматривается как «вычислительный молекулярный микроскоп» — позволяет исследовать молекулярные явления с атомным разрешением.

## ❓ Вопрос 2: Силовые поля в МД-симуляциях: типы, преимущества и недостатки

> [!tip] Ключевые тезисы
> - Силовое поле = набор параметров для описания ковалентных и нековалентных взаимодействий
> - Основные члены: связи, углы, торсионы, электростатика (Кулон), Ван-дер-Ваальс (Леннард-Джонс)
> - Нет универсального поля: для белков, лигандов, углеводов, мембран — разные поля
> - Модели воды: TIP3P (быстрая, стандартная), OPC (точная, с ff19SB)

### Развёрнутый ответ

**Что такое силовое поле?**
Силовое поле — это математическое описание всех межатомных взаимодействий в системе. Это набор уравнений и параметров, сохранённых в файле. Основные энергетические члены:

- **Ковалентные (bonded):** валентные связи (гармонический потенциал), углы (гармонический потенциал), торсионные углы (периодический потенциал — правильные и неправильные диэдры)
- **Нековалентные (non-bonded):** электростатические взаимодействия (потенциал Кулона: q₁q₂/r) + Ван-дер-Ваальс (потенциал Леннарда-Джонса: 4ε[(σ/r)¹² − (σ/r)⁶])

**Основные семейства силовых полей:**

**AMBER:** ff14SB, ff19SB, ff99SB — для белков; GAFF/GAFF2 — для органических лигандов; GLYCAM06 — для углеводов и циклодекстринов; Lipid21 — для мембран.
- *ff14SB:* улучшенные параметры боковых цепей, воспроизводит Рамачандрановские распределения; используется с TIP3P водой
- *ff19SB:* новейший, улучшенные φ/ψ-торсионные параметры на основе квантовых данных; рекомендуется с OPC водой
- *ff99SB:* наиболее протестированный, но устаревший; подходит для проверки на уже опубликованных системах

**CHARMM36:** широко применяется с NAMD; хорошо параметризован для белков, нуклеиновых кислот, липидов; поддерживается CHARMM-GUI

**GROMOS:** исторически развит в Европе (Гронинген, Цюрих); открытый исходный код; применяется в GROMACS; унифицированный атомный форматизм

**OPLS-AA / OPLS3:** хорошо описывает жидкости и фармацевтические соединения; часто используется в Desmond (Schrödinger)

**Модели воды:**
- **TIP3P:** наиболее распространённая; быстрая; используется с ff99SB и ff14SB; хорошо воспроизводит динамику в стандартных условиях
- **SPC/E:** улучшенная; лучше воспроизводит диффузию и диэлектрическую проницаемость
- **TIP4P-Ew / TIP4P/2005:** четырёхточечные модели; более точные структурные свойства
- **OPC:** современная четырёхточечная модель; рекомендуется с ff19SB; лучшее воспроизведение свойств воды

**Преимущества силовых полей:**
- Быстрые вычисления (классическая механика Ньютона, не квантовая)
- Позволяют симулировать большие системы (десятки–сотни тысяч атомов, мкс-траектории)
- Хорошо тестированы и валидированы для стандартных биомолекул

**Недостатки:**
- Могут создавать **артефакты** (неверные конформации) при плохой параметризации
- Требуют тщательной параметризации для нестандартных молекул (металлические ионы, ковалентно связанные лиганды)
- Не учитывают квантовые эффекты (разрыв/образование связей невозможно смоделировать)
- Разные поля дают разные результаты → сложно сравнивать данные разных групп

**Классическая vs. Ab initio МД:**
| Параметр | Классическая МД | Ab initio МД |
|---|---|---|
| Основа | F = ma (Ньютон) | Квантовая химия (DFT) |
| Размер системы | до ~10⁶ атомов | ~сотни атомов |
| Временной масштаб | нс – мкс | пс |
| Точность | умеренная | высокая |
| Скорость | быстро | медленно |

## ❓ Вопрос 3: Ключевые параметры МД-симуляций для оценки белка

> [!tip] Ключевые тезисы
> - **RMSD** → стабильность (отклонение от начальной структуры)
> - **RMSF** → гибкость (по остаткам, сравнивается с B-фактором)
> - **Rg** → компактность (свёрнутость) белка
> - **Водородные связи** → стабильность комплекса
> - **MM-PBSA/MM-GBSA** → свободная энергия связывания (ΔG)

### Развёрнутый ответ

**RMSD (Root Mean Square Deviation)**
Среднеквадратичное отклонение — основной параметр для оценки **стабильности** белка или комплекса:
RMSD = √(Σᵢ mᵢ|rᵢ(t) − rᵢ(ref)|² / Σᵢ mᵢ)
- Если RMSD выходит на плато → система достигла равновесия
- Высокий RMSD = большие конформационные изменения, нестабильность
- Если RMSD не стабилизируется → нужно продолжать симуляцию (обычно ≥ 100–500 нс)
- Рассчитывается в CPPTRAJ (AMBER) или gmx rms (GROMACS)

**RMSF (Root Mean Square Fluctuation)**
Отражает **гибкость** каждого аминокислотного остатка:
- Высокое RMSF → подвижные петли, N- и C-концевые участки
- Низкое RMSF → жёсткие вторичные структуры (α-спирали, β-листы)
- Напрямую сравнимо с **B-фактором** из рентгеноструктурного анализа

**B-фактор (температурный фактор)**
Экспериментальный параметр из кристаллографии (хранится в PDB-файлах):
- < 30 Å² → высокое доверие к положению атомов
- > 60 Å² → высокий беспорядок, белок сложно моделировать
- Позволяет **валидировать** МД-данные: RMSF из симуляции должен коррелировать с B-фактором

**Радиус вращения (Radius of Gyration, Rg)**
Характеризует **компактность** (свёрнутость) белка:
- Низкий Rg = глобулярная структура (нативная, свёрнутая)
- Растущий Rg = разворачивание белка (unfolding)
- Используется совместно с RMSD

**Водородные связи (Hydrogen Bonds)**
Число и длительность водородных связей:
- Белок↔лиганд: оценка стабильности комплекса (больше связей → стабильнее)
- Белок↔растворитель: оценка степени сольватации
- Внутри белка: определяет вторичную структуру

**MM-PBSA / MM-GBSA**
Постпроцессинг МД-траекторий для расчёта **свободной энергии связывания**:
ΔG_bind = ΔE_MM + ΔG_solvation − TΔS
- MM-PBSA (Poisson-Boltzmann) — более точный
- MM-GBSA (Generalized Born) — быстрее, чаще используется для ранжирования лигандов
- Точнее, чем оценочные функции молекулярного докинга

**SASA (Solvent Accessible Surface Area)**
Площадь поверхности, доступной растворителю:
- Изменения SASA при связывании → гидрофобный вклад
- Используется в расчётах MM-PBSA

**PCA (Principal Component Analysis)**
Анализ главных компонент траектории:
- Выявляет доминирующие моды движения белка
- Позволяет строить Марковские модели состояний (MSM)
- Определяет коллективные движения, связанные с функцией

**Общий протокол анализа:**
1. Минимизация энергии (energy minimization)
2. Нагрев (heating) → NVT (постоянный объём/температура)
3. Уравновешивание → NPT (постоянное давление/температура)
4. Продуктивная симуляция (100 нс – 1 мкс)
5. Расчёт: RMSD → RMSF → Rg → H-bonds → MM-PBSA → PCA

## 📄 Статья: Ergodicity Breaking and Self-Destruction of Cancer Cells

> [!summary] Краткий обзор для экзамена
> Shityakov et al. (2023) предложили энтропийную модель геномного хаоса при раке. Хромотрипсис — катастрофическая геномная перестройка — увеличивает скорость мутаций на три порядка (до 10⁻⁴). При превышении порога ~10% клеток с хромотрипсисом вероятность мутации BCL2 стремится к 1, что запускает апоптоз (самоуничтожение) опухолевых клеток. Это подтверждено экспериментально на рабдомиосаркоме RA-2 крыс и вычислительным Монте-Карло моделированием мутаций BCL2. Ключевая идея: искусственное усиление геномного хаоса — потенциальная противораковая стратегия.

> [!note] Подробный анализ
> Полный разбор статьи: [[Shityakov_2023_Ergodicity]]

### Ключевые факты для экзамена
- **Авторы:** Shityakov S., Kravtsov V., Skorb E.V., Nosonovsky M. (ИТМО, СПб)
- **Журнал:** Entropy (Basel), 2024, 26(1), 37
- **Объект:** рабдомиосаркома RA-2 крыс, ген BCL2
- **Метод:** Монте-Карло симуляция мутаций (Python), гомологичное моделирование (SWISS-MODEL), протокол Rosetta (стабильность), ZDOCK (докинг белок-белок), PPI-affinity (энергия связывания)
- **Маркер геномного хаоса:** FCM (частота клеток с микроядрами)
- **Порог апоптоза:** FCM ≈ 10% → вероятность мутации BCL2 → 1
- **Ключевой результат:** ΔES = +4,11 REU (дестабилизация мутанта G5W); Ebind мутанта = −11,1 ккал/моль (хуже, чем WT −11,3 ккал/моль)
- **Практический вывод:** индукция геномного хаоса как противоопухолевая стратегия; риск: развитие лекарственной устойчивости
- **Связь с хемоинформатикой:** докинг, гомологичное моделирование, Монте-Карло, Python-скриптинг, базы данных COSMIC/PDB
