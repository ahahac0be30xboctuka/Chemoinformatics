## Вопрос 1: Два основных in silico метода и их отличия

> [!tip] Ключевые тезисы
> - **Ligand-based** — работает без структуры белка, быстрый, скринирует миллионы
> - **Structure-based** — требует 3D-структуры мишени, точный, ресурсоёмкий
> - В воронке скрининга: сначала ligand-based (млн → тысячи), потом structure-based (тысячи → единицы)
> - Фармакофор — гибрид между двумя подходами

**Ligand-based design** — опирается на известные активные молекулы. Структура белка не нужна — анализируются свойства лигандов. Инструменты: **фильтры** (правило Липинского: MW < 500, LogP < 5, TPSA < 140 Å²), **поиск по сходству** через молекулярные отпечатки, **QSAR-модели** (связывают дескрипторы с активностью) и **фармакофорное моделирование** (3D-паттерн: донор Н-связи, акцептор, гидрофобный центр). Главное достоинство — скорость: миллионы соединений за несколько дней.

**Structure-based design** требует 3D-структуры мишени (рентгеноструктурный анализ, ЯМР). Главный инструмент — **молекулярный докинг**: вычисление аффинности гибкого лиганда к активному сайту рецептора через **скоринг-функции**. Принцип «ключ–замок» Эмиля Фишера. Даёт точные позы связывания и энергии взаимодействия, но значительно медленнее.

В **воронке виртуального скрининга** оба метода работают последовательно: ligand-based сужает пространство кандидатов, structure-based проводит финальный точный отбор. Именно такая двухуровневая схема позволяет сбалансировать скорость и качество.

## Вопрос 2: Почему исследование ГЭБ критически важно для хемоинформатики?

> [!important] Ключевые тезисы
> - **98% веществ** не проникают через ГЭБ — главный фильтр для CNS-препаратов
> - Для CNS-препаратов проницаемость обязательна; для периферических — нежелательна
> - Эксперимент дорог → нужны быстрые in silico предсказания
> - Ключевые пороги: **TPSA < 60–70 Å²**, **MW < 400–600 Да**, **LogP** оптимален
> - **P-гликопротеин (P-gp)** — активный эффлюксный транспортёр, обходит пассивные модели

**Гематоэнцефалический барьер (ГЭБ)** — полупроницаемый барьер между кровью и мозгом, образованный эндотелием с плотными контактами (claudin, occludin). Он не пускает **98% всех потенциальных лекарств**, что делает ГЭБ одним из главных препятствий в разработке CNS-препаратов (болезни Альцгеймера, Паркинсона, опухоли мозга).

Экспериментальная оценка — дорогая и медленная (опыты на животных, трансвелл-системы, ПЭТ). Поэтому хемоинформатика предлагает **раннее in silico предсказание** ещё до синтеза.

Молекула преодолевает ГЭБ пассивно (диффузия через мембрану, зависит от полярности) или активно (транспортёры). Ключевой параметр — **logBB = log(C_мозг/C_кровь)**. Дескрипторы-предикторы: TPSA < 60–70 Å², LogP (липофильность), MW < 400–600 Да. **P-gp** (P-гликопротеин) — эффлюксный насос, который выкачивает лекарства обратно из мозга. Молекулы-субстраты P-gp не подчиняются пассивным моделям и становятся «выбросами» — это нужно учитывать при моделировании.

## Вопрос 3: Как хемоинформатика использует экспериментальные данные для обучения?

> [!summary] Ключевые тезисы
> - Экспериментальные данные — обязательная основа для обучения и **валидации** любой модели
> - **Индуктивное** обучение: данные → закономерность (QSAR, logBB → модель)
> - **Дедуктивное** обучение: знание → предсказание данных (NeuroClick, механистические модели)
> - Валидация: **R²**, **RMSE**, **Q²**, кросс-валидация 80/20, RMSD < 2 Å для докинга

Хемоинформатика **не существует без экспериментальных данных** — они нужны для обучения и проверки моделей. Концептуальная пирамида: сырые измерения (IC₅₀, logBB, Ki) → контекстуализированная информация → обобщённое **знание** (правила, модели).

**Индуктивное обучение** (данные → закономерность) — основа большинства ML-моделей: QSAR обучается на экспериментальных IC₅₀, модель ГЭБ — на экспериментальных logBB, предсказание **LogP** (метод Molinspiration) подогнано на > 12 000 измерений.

**Дедуктивное обучение** (знание → данные): используется известный механизм реакции для генерации предсказаний. Пример — **NeuroClick**: знание о механизме клик-реакции → генерация базы триазольных соединений → экспериментальная верификация.

**Валидация** без исключений обязательна: разбивка **80% / 20%** тренировочный/тестовый набор с 5-кратной кросс-валидацией, метрики R² и Q² (кросс-валидационный R²), RMSE. Для докинга — RMSD < 2 Å относительно кристаллической структуры. Цикл «эксперимент → модель → предсказание → новый эксперимент» — двигатель рационального дизайна лекарств.

## 📄 Статья: Shityakov et al. (2013) — PSA-дескрипторы для предсказания проницаемости ГЭБ

> [!abstract] Карточка статьи
> **Цель:** Сравнить 4 дескриптора полярной поверхности (2D-PSA, TPSA, 3D-PSA, W2) по качеству предсказания проницаемости ГЭБ.
>
> **Метод:** PLS-регрессия на 19 CNS-активных/неактивных препаратах; 3D-PSA рассчитана квантово-механически (DFT/6-31G*); программы Spartan'10, MOE2009, Volsurf+.
>
> **Главный результат:** **3D-PSA → R² = 0.92, RMSD = 0.29** (лучший результат); TPSA → R² = 0.85; 2D-PSA → R² = 0.71; W2 → R² = 0.42.
>
> **Вывод:** Полярные дескрипторы эффективны для раннего скрининга, но только при пассивной диффузии. P-gp делает ряд молекул выбросами.

> [!note] Факты, которые точно спросят
> - **Пороги:** PSA < 60 Å² → проникает через ГЭБ; PSA > 120–140 Å² → не проникает; W2 > 750 Å³ → не проникает
> - **Выбросы** (субстраты P-gp): icotidine, domperidone, risperidone, 9-hydroxyrisperidone — выкачиваются из мозга активным транспортом, поэтому не вписываются в пассивную модель
> - **TPSA vs 3D-PSA:** TPSA быстрее в расчёте, 3D-PSA точнее (квантовая химия). Для быстрого скрининга достаточно TPSA
> - **logBB** = log(C_мозг / C_кровь) — единственный количественный показатель проницаемости ГЭБ в работе
