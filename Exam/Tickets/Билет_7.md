## ❓ Вопрос 1: Алгоритмы protein–protein молекулярного докинга. Отличия от protein–ligand докинга

> [!tip] Ключевые тезисы
> - PPI докинг работает с двумя крупными партнёрами → глобальный поиск по всей поверхности (не в grid box)
> - Основные алгоритмы: FFT (ZDOCK, ClusPro), полуфлексибельный (HADDOCK), Monte Carlo (RosettaDock)
> - Ключевые отличия: размер партнёров, масштаб поискового пространства, характер интерфейса, вычислительная стоимость

### Развёрнутый ответ

**Protein–protein interaction (PPI) докинг** — вычислительный метод предсказания пространственной организации комплекса двух белков. Применяется в хемоинформатике и синтетической биологии для изучения механизмов сигнализации, выявления мишеней и дизайна пептидных ингибиторов.

#### Основные алгоритмы PPI докинга

**1. FFT-методы (быстрое преобразование Фурье)**

Используются в **ZDOCK** и **ClusPro**. Принцип: белки представляются как трёхмерные сетки; FFT позволяет за одно преобразование оценить сродство всех возможных трансляций/вращений одного белка относительно другого. Это «глобальный» поиск — сканируется вся поверхность.

- **ZDOCK**: rigid-body FFT, оценивает электростатику + форму поверхности + десольватацию.
- **ClusPro**: FFT-сэмплинг (~70 000 поз) → кластеризация → минимизация энергии кластерных центров.

**2. HADDOCK** (High Ambiguity Driven protein–protein DOCKing)

Полуфлексибельный метод, уникальный тем, что использует **экспериментальные ограничения** (данные NMR, сайт-направленного мутагенеза, масс-спектрометрии) в виде «расплывчатых взаимодействий» (ambiguous interaction restraints). Три стадии: жёсткий докинг → полуфлексибельная минимизация → уточнение в явной воде. Считается одним из наиболее точных методов при наличии экспериментальных данных.

**3. RosettaDock**

Алгоритм на основе **Monte Carlo с локальной минимизацией**. Учитывает гибкость боковых цепей на интерфейсе (side chain repacking). Использует силовое поле Rosetta (комбинация физических и статистических потенциалов). Применяется для предсказания интерфейсов с высокой точностью.

**4. PatchDock**

Геометрическое сопоставление поверхностей (geometric complementarity matching). Ищет комплементарные «заплатки» на поверхностях двух белков. Быстрый, но менее точный.

#### Принципиальные отличия PPI от protein–ligand докинга

| Характеристика | Protein–ligand | Protein–protein |
|---|---|---|
| **Размер «лиганда»** | Малая молекула (~70 атомов) | Белок (~сотни–тысячи остатков) |
| **Поисковое пространство** | Ограниченный grid box (20 Å) | Вся поверхность (~сотни Å²) |
| **Алгоритм поиска** | Генетический алгоритм (AutoDock), BFGS (Vina) | FFT, Monte Carlo, геометрическое сопоставление |
| **Гибкость** | Гибкий лиганд + жёсткий рецептор | Оба партнёра могут быть гибкими |
| **Интерфейс связывания** | Компактный карман | Протяжённый плоский интерфейс |
| **Scoring** | Force field, empirical, ML | Комбинированные функции + кластеризация |
| **Вычислительная стоимость** | Минуты (CPU) | Часы–дни (кластер) |
| **Сложность валидации** | RMSD относительно кристаллической позы | Трудно без экспериментальных ограничений |

**Применение в курсе (пример из Лекции 7):** анализ взаимодействия HIV-интегразы с транспортином Sr. Использовался гибкий докинг петли H8, были идентифицированы ключевые остатки интерфейса (Arg33/36, Asp328, Lys161). Итоговый комплекс включал 2 молекулы транспортина + 2 молекулы интегразы + вирусная ДНК.

## ❓ Вопрос 2: Молекулярная динамика — определение, типы, важность для хемоинформатики, преимущества и недостатки

> [!tip] Ключевые тезисы
> - МД = численное решение уравнений Ньютона (F = ma) для N-тельной системы
> - Детерминированный метод; 4 типа: классическая, ab initio, all-atom, coarse-grained
> - Нобелевская премия 1999 — Карплус, Левит, Варшель
> - МД = «молекулярный микроскоп» (Michael Levitt)

### Развёрнутый ответ

**Молекулярная динамика (МД)** — численный метод симуляции физических движений атомов и молекул в N-тельной системе. МД решает уравнения движения Ньютона (**F = ma**) для каждого атома на каждом временно́м шаге, что позволяет отслеживать эволюцию системы во времени. Метод **детерминированный**: зная начальные положения и скорости, можно точно предсказать будущее состояние.

Метод разработан в 1950-х годах (Розенблют, Метрополис, США) параллельно с методом Монте-Карло. В 1999 г. Нобелевская премия по химии была вручена Карплусу, Левиту и Варшелю за разработку многомасштабных методов моделирования. Майкл Левит охарактеризовал МД как «молекулярный микроскоп», позволяющий исследовать явления, недоступные экспериментальным методам.

#### Типы молекулярной динамики

**1. Классическая МД**
Взаимодействия описываются эмпирическими силовыми полями (force fields). Уравнения Ньютона решаются численно с шагом ~2 фемтосекунды. Силовое поле содержит: энергию связей, углов, диэдральных углов, Ван-дер-Ваальса (потенциал Леннарда–Джонса) и электростатику (кулоновский потенциал).
- Основные силовые поля: **ff99SB**, **ff14SB**, **ff19SB** (AMBER) для белков; GLYCAM06 для углеводов; lipid14 для мембран; GAFF для лигандов; CHARMM36; OPLS.
- Модели воды: TIP3P, SPC/E, TIP4P, OPC.
- Временно́й масштаб: фс – мкс.

**2. Ab initio МД (квантово-механическая)**
Электронная структура рассчитывается на каждом шаге из первых принципов (DFT или HF). Высокая точность, но ограничена ~сотнями атомов и пикосекундами траектории. Программа: TERACHEM (GPU).

**3. All-atom МД**
Все атомы системы явно представлены, включая атомы водорода. Максимальная детализация. Применима при наличии HPC-ресурсов (GPU-кластеры).

**4. Coarse-grained МД (огрублённая)**
Группы атомов объединяются в «бусины» (beads). Например, молекула из 138 атомов → 15 узлов. Позволяет моделировать большие системы (белковая агрегация, сворачивание) и длинные траектории (мкс–мс).

**5. QM/MM (гибридная)**
Сайт связывания (активный центр) — квантово-механически (DFT); остальная система — классически. Компромисс точность/скорость. Реализована в AMBER (через внешние интерфейсы).

#### Важность для хемоинформатики

МД используется в хемоинформатическом пайплайне:
- **Постдокинговое уточнение:** подтверждение стабильности белок-лигандных комплексов (как в статье Kovalenko 2023 — 200 нс AMBER МД для D3R).
- **Расчёт свободных энергий:** MM-PBSA, MM-GBSA (быстрые) и FEP (точные, дороже).
- **LIE (Linear Interaction Energy):** ΔG_lie = α⟨ΔE_vdW⟩ + β⟨ΔE_elec⟩ — использован в Kovalenko 2023.
- **Предсказание BBB-проницаемости:** симуляция в мембранном окружении.
- **Белковое сворачивание и агрегация:** болезнь Альцгеймера (амилоидные агрегаты), динамика флуктуаций (RMSF).
- **Параметры гибкости:** RMSD (отклонение от стартовой структуры), RMSF (флуктуации остатков), радиус гирации.

#### Программное обеспечение

| ПО | Особенности |
|---|---|
| **GROMACS** | Белок–белковые системы, мультибелковые комплексы, бесплатный, инструменты анализа |
| **AMBER** | Лучший для protein–ligand, GPU (pmemd.cuda), AMBER Tools 22/23 бесплатно, MM-PBSA/GBSA |
| **NAMD** | Нанобиотехнологии, VMD-интерфейс, бесплатный |
| **DESMOND** | 60–80× быстрее CPU, FEP-модуль, коммерческий (Schrödinger) |

#### Преимущества и недостатки

**Преимущества:**
- Детализация переходных состояний и динамики
- Количественный расчёт ΔG связывания
- Детерминированность и воспроизводимость
- Применимость к белкам, нуклеиновым кислотам, мембранам, полимерам

**Недостатки:**
- Требует HPC (кластеры, GPU) — для простого докинга достаточно CPU
- Временны́е ограничения: биохимические процессы (мс – с) требуют огрублённых моделей
- Проблемы сэмплинга: система может «застревать» в локальных минимумах
- Качество зависит от параметризации силового поля
- Классическая МД: нет информации об электронах и зарядовом переносе

## ❓ Вопрос 3: Химические расчёты в хемоинформатике для рационального дизайна лекарств

> [!tip] Ключевые тезисы
> - Иерархия: квантовая химия (DFT) → классическая механика (МД, докинг) → хемоинформатика (дескрипторы)
> - ADME/ADMET — ключевые фармакокинетические фильтры (Lipinski, Veber, Egan)
> - QSAR/QSPR — количественные соотношения структура–активность/свойства
> - logBB = f(ClogP, TPSA) — расчёт BBB-проницаемости

### Развёрнутый ответ

Хемоинформатика использует три уровня теоретических методов для рационального дизайна лекарств:

#### 1. Квантовая химия (DFT, ab initio)

**DFT (теория функционала плотности)** — наиболее точный и вычислительно доступный квантово-механический метод. Оперирует электронами и ядрами, описывает волново-корпускулярный дуализм.

Применения в дизайне лекарств:
- **Электростатические потенциальные карты** — визуализация распределения заряда на поверхности молекулы, понимание взаимодействий с рецептором.
- **3D полярная поверхность (3D-PSA)** — уточнённый расчёт BBB-проницаемости с учётом конформации.
- **HOMO–LUMO Gap** — коррелирует с липофильностью и биологической активностью; важен для транзисторных материалов и ингибиторов.
- **Конформационная оптимизация** — минимизация энергии молекулы.

Ограничения: применима только к небольшим органическим молекулам (лигандам); не масштабируется на сотни конформеров.

#### 2. Молекулярный докинг и динамика (классическая механика)

Расчёты на основе силовых полей:
- **Молекулярный докинг** — расчёт аффинности связывания белок–лиганд (ккал/моль); поиск оптимальной позы в сайте связывания.
- **МД** — симуляция динамики комплекса; расчёт ΔG_bind через MM-PBSA/MM-GBSA/LIE/FEP.

#### 3. Расчёт молекулярных дескрипторов и ADME/ADMET

**Дескрипторы** — числовые представления молекулярных свойств (~5000 типов):
- **ClogP** (lipophilicity) — октанол-водный коэффициент распределения
- **TPSA** (topological polar surface area) — полярная поверхность, предсказывает всасывание и BBB-проницаемость
- **MW** (molecular weight) — молекулярная масса
- **HBD/HBA** — доноры/акцепторы водородных связей
- **Number of rotatable bonds** — гибкость молекулы

**ADME/ADMET предсказания** — оценка фармакокинетики без эксперимента:
- **Absorption (A):** биодоступность, проницаемость кишечника
- **Distribution (D):** объём распределения, BBB-проницаемость (logBB)
- **Metabolism (M):** сайты метаболизма (CYP450)
- **Excretion (E):** почечный клиренс
- **Toxicity (T):** канцерогенность, мутагенность (правила Бениньи–Россы, Крамера)

**Ключевые правила-фильтры:**
- **Правило Липинского (Rule of Five):** MW < 500 Да, logP < 5, HBD ≤ 5, HBA ≤ 10 — предсказывает пероральную биодоступность
- **Правило Вебера:** ≤ 10 вращающихся связей, TPSA ≤ 140 Å²
- **Правило Игана:** logP и TPSA попадают в «эйплипейс»

**Расчёт BBB-проницаемости (logBB):**
$$\text{logBB}_{Clark} = 0.152 \times ClogP - 0.0148 \times TPSA + 0.139$$
$$\text{logBB}_{Rishton} = 0.155 \times ClogP - 0.01 \times TPSA + 0.164$$

Порог: **logBB ≥ 0.3** — молекула хорошо проникает через BBB.

**PAINS (Pan Assay Interference Compounds)** — фильтрация соединений, дающих ложноположительные результаты в HTS (высокопроизводительном скрининге). Примеры: тоскофлавин, изолины, хиноны.

#### 4. QSAR/QSPR

**Количественные соотношения структура–активность/свойства** — математические модели, связывающие молекулярные дескрипторы с биологической активностью или физико-химическими свойствами.

- **QSAR**: предсказывает IC50, Ki, % ингибирования и т.д. из дескрипторов
- **3D-QSAR**: использует трёхмерную структуру (CoMFA, CoMSIA)
- Методы: множественная регрессия, нейронные сети, случайный лес
- Применение: обогащение датасетов для виртуального скрининга; предсказание активности несинтезированных молекул

**Пример из статьи Kovalenko 2023:** logBB — типичный QSPR-дескриптор, рассчитываемый из ClogP и TPSA; используется как фармакокинетический фильтр для отбора BBB-проницаемых ингибиторов D3R.

## 📄 Статья: Kovalenko et al. 2023 — Click chemistry для D3R ингибиторов

> [!summary] Краткий обзор для экзамена
> Авторы разработали новый in silico workflow: клик-реакция (RDKit, SMARTS-паттерн) → библиотека 18 838 триазолов → фильтрация по logBB (Clark/Rishton) → молекулярный докинг (AutoDock Vina + iDock) → идентификация хита ID 7000 → 200-нс МД (AMBER, мембрана POPC:POPE:холестерин). Хит превзошёл референс этиклоприд по аффинности (−10.16 vs −7.64 ккал/моль Vina) и BBB-проницаемости (logBB 0.34 vs −0.35). МД подтвердила: ΔG_lie = −60.17 vs −58.37 ккал/моль за счёт Ван-дер-Ваальса.

> [!note] Подробный анализ
> Полный разбор статьи: [[Kovalenko_2023_D3R_ClickChemistry]]

### Ключевые факты для экзамена

- **Мишень:** Рецептор дофамина D3 (D3R) — G-белок-сопряжённый рецептор, мишень при шизофрении, Паркинсоне, наркомании
- **Метод:** In silico клик-реакция (1,3-диполярное циклоприсоединение азида к алкину) → 1,4- и 1,5-триазолы; RDKit, SMARTS
- **Скаффолд:** 1-phenyl-4-[4-(1H-1,2,3-triazol-5-yl)butyl]piperazine (триазол = биоизостер амида)
- **Библиотека:** 143 075 азидов (PubChem) → 18 838 триазолов после фильтрации Липинского
- **BBB-фильтр:** logBB ≥ 0.3; формулы Clark и Rishton (функции ClogP и TPSA)
- **Докинг:** AutoDock Vina v1.1.2 + iDock v2.2.3; структура D3R — PDB: 3PBL (с этиклопридом)
- **Лучший хит:** ID 7000 (1,5-изомер); Vina −10.16, iDock −10.17 ккал/моль; logBB Clark 0.34
- **МД:** AMBER 20, 200 нс, мембрана POPC:POPE:холестерин (100:100:50), силовые поля ff99SB + lipid14 + GAFF
- **Главный результат:** ΔG_lie = −60.17 ккал/моль (хит) vs −58.37 ккал/моль (этиклоприд); превосходство за счёт ΔE_vdW < −50 ккал/моль
- **Вывод:** Workflow клик-реакция + logBB-фильтрация применим для разработки антипсихотиков
