## Вопрос 1: Цель хемоинформатики. Классификация методов по точности

**Цель хемоинформатики** — сузить химическое пространство от ~10⁶⁰ возможных молекул до нескольких кандидатов с нужными свойствами, используя вычислительные методы.

**Как молекула попадает в химическое пространство:**
Берёшь молекулу → считаешь N дескрипторов (MW, logP, TPSA, ...) → получаешь вектор из N чисел → это координаты точки в N-мерном пространстве.
```
Молекула А: [500, 3.2, 120, ...] → точка
Молекула Б: [480, 2.8, 115, ...] → другая точка
```
**Расстояние между точками = мера похожести.** Чем ближе → тем похожее молекулы по свойствам:
```
d = √((500-480)² + (3.2-2.8)² + (120-115)² + ...)
```
Проблема: разные единицы и масштабы (MW — сотни, logP — единицы) → перед расчётом обязательна **нормализация** дескрипторов.

**Операции в химическом пространстве:**
- **Поиск по сходству** — найти молекулы вблизи заданной (similarity search)
- **Кластеризация** — сгруппировать близкие молекулы
- **Навигация** — двигаться от хита к лиду, оптимизируя свойства

Вычислительные методы по точности и скорости:

1. **DFT (квантовая химия)** — самые точные, считают электронную структуру. Только для малых молекул, очень медленно.
2. **Силовые поля (классическая механика)** — потенциалы Леннарда-Джонса и т.п. Быстрее, применяются в молекулярной динамике.
3. **Дескрипторы + ML/DL**:
   - *Классический*: дескрипторы + мл. регрессия
   - *ML/DL*: дескрипторы/fingerprints/граф + Random Forest, GNN, нейросети

По типу: лиганд-базированные (нет структуры белка) и структурно-базированные (есть 3D — докинг). На практике сначала быстрые фильтры, потом докинг.

## Вопрос 2: Метрики (Python) в хемоинформатике

Модели решают две задачи: **классификацию** (активная молекула или нет) и **регрессию** (предсказать числовое значение, например IC₅₀).

Метрики классификации:
- **ROC-AUC** — площадь под кривой ошибок; 1.0 идеально, 0.5 — случайное угадывание.
- **BEDROC** — модификация ROC-AUC: штраф, если активные молекулы в конце списка (важнее топ).
- **EF (Enrichment Factor)** — доля активных молекул в топ X% списка → улучшенная классификация.
- **Precision, Recall, F₁**

Метрики регрессии (предсказать значение):
- **RMSE** — среднеквадратичная ошибка
- **R²** — доля объяснённого разброса (1.0 = идеал)
- **Q²-R²** — Q² = R² на кросс-валидации; разрыв Q² < R² означает переобучение

Стандарт: 80% обучение / 20% тест, 5-кратная кросс-валидация. Библиотеки: `sklearn` (метрики, алгоритмы), `RDKit` (дескрипторы и фингерпринты).

## Вопрос 3: Преимущества и недостатки молекулярных скаффолдов

**Скаффолд** (scaffold) — общее ядро структуры для серии соединений. Заместители варьируют для оптимизации активности.

**Задачи со скаффолдами:**
- **Scaffold Hopping** — меняется каркас, но сохраняется биоактивности (замена фрагмента по эл.-хим. свойствам) → расширение хим. пространства
- **Scaffold Diversity** — собрать коллекцию молекул с максимально разными каркасами → максимально разное хим. пространство
- **Scaffold Murcko** — извлечение каркаса из молекулы (метод в RDKit) → группировка молекул по каркасу *(инструмент)*

Преимущества скаффолдного подхода: организация БД, репозиционирование препаратов, генерация библиотек.

Недостатки: сужение хим. пространства, синтетические ограничения, наследование плохих свойств каркаса.

## Статья: Shityakov & Förster (2014) — модель транспорта через ГЭБ

**Зачем изучали:** ~98% лекарств не проникают в мозг; нужен способ предсказать, какие молекулы доставит туда холиновый транспортёр ГЭБ (BBB-ChT — белок-переносчик холина через гематоэнцефалический барьер).

**Как делали:** BBB-ChT не имеет 3D-структуры — построили гомологичную модель по шаблону натрий/галактозного симпортера (PDB: 3DH4). Провели докинг (AutoDock и AutoDock Vina) 93 соединений; порог активности — ΔGbind < −6.0 ккал/моль.

**Что получили:** r²=0.88 (предсказанные энергии vs эксперимент), AUC=0.82 — модель уверенно разделяет активные и неактивные молекулы. Артефакт: молекулы с <20 тяжёлыми атомами дают ложноотрицательный докинг.

**Вывод:** Гомологичная модель + докинг — рабочая альтернатива QSAR (предсказание активности по структуре) при отсутствии кристаллической структуры белка.

> [!important] Связь с вопросами билета
> Статья иллюстрирует структурно-базированный подход (Вопрос 1) и применение метрик AUC и r² для оценки модели (Вопрос 2); скаффолды бис-пиридиниевых циклофанов служат примером целенаправленного использования молекулярного каркаса для ЦНС-доставки (Вопрос 3).
