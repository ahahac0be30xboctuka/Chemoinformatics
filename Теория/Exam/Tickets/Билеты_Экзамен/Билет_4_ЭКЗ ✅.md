## Вопрос 1: Преимущества и недостатки концепции фармакофора. Типы фармакофоров.

**Фармакофор** — абстрактная 3D-модель того, какие хим. группы и где именно в пространстве должны быть у молекулы, чтобы она связалась с белком. Это **не конкретная молекула, а шаблон требований**.

*Пример:* H-donor на расстоянии 5 Å, отрицательный заряд рядом, ... → любая молекула, отвечающая этому шаблону, потенциально активна.

**Плюсы (+):**
- Ligand-based → без структуры белка
- Быстрый скрининг (проверка на соответствие фармакофору, не докинг)
- Объясняет, почему структурно разные молекулы активны
- Различает близкородственные мишени (D2/D3 дофамина, 78% идентичность)

**Минусы (−):**
- Не учитывает гибкость белка
- Не учитывает сольватацию и сопутствующие эффекты
- Требует набора известных активных молекул

**Типы:**
1. Признаковые (feature-based) — набор конкретных химических групп
2. Нечёткие (fuzzy) — учитывают пространственную неопределённость
3. Лиганд-ориентированные — выводятся из анализа активных молекул
4. Структурно-ориентированные — выводятся из 3D-структуры сайта связывания белка
5. Полевые (field-based) — учитывают молекулярные поля; применяются в CoMFA/CoMSIA-анализе (сравнительный молекулярный анализ полей для построения QSAR)

## Вопрос 2: Молекулярный докинг в хемоинформатике. Типы молекулярного докинга.

**Молекулярный докинг** — вычисление аффинности связывания рецептора (белка) и лиганда. Алгоритм перебирает конформации лиганда в сайте связывания и ищет позу с минимальной свободной энергией Гиббса (ΔG).

**Применяется для:** виртуального скрининга библиотек, оптимизации лид-соединений, поиска новых показаний для известных препаратов (drug repurposing), предсказания побочных эффектов.

**Типы по гибкости системы:**
1. Жёсткий рецептор / гибкий лиганд (rigid-flexible) — стандарт, быстрый
2. Оба гибкие (flexible-flexible) — реалистичнее, вычислительно дорог
3. Оба жёсткие (rigid-rigid) — геометрическое сопоставление поверхностей
4. Избирательная гибкость — гибкие только отдельные остатки рецептора

**По типу системы:** белок-лиганд, белок-белок, хозяин-гость (host-guest, например циклодекстрины).

**Ключевые ограничения:** рецептор считается жёстким, вода в сайте не учитывается, стандартная ошибка до 1.5 ккал/моль.

**Скоринговая функция** — оценка выгодности позы лиганда в активном сайте белка, выражена в ккал/моль. **4 класса:**

1. **Силовые поля (Force field-based, FF)** — суммируют межмолекулярные взаимодействия рецептора и лиганда:
   `score = E_VdW + E_elec`
   - E_VdW = ε[(r_min/r)¹² − 2(r_min/r)⁶] (потенциал Леннарда-Джонса)
   - E_elec = q₁q₂/εr (закон Кулона)
   - Применение: AutoDock 4

2. **Knowledge-based (статистические потенциалы)** — основаны на статистике межмолекулярных контактов в больших 3D-базах данных (PDB). Логика: если пара атомов встречается в кристаллических структурах чаще случайного → взаимодействие выгодно.
   - Применение: PMF-score (PatchDock), DrugScore

3. **Эмпирические** — подсчёт числа различных типов взаимодействий; веса подбираются по экспериментальным данным:
   `score = w₁(H-bonds) + w₂(гидрофобность/SASA) + w₃(гибкость лиганда) + w₄(электростатика) + ...`
   - Применение: AutoDock Vina, GlideScore, ChemScore

4. **Гибридные (Hybrid)** — комбинация FF + эмпирические (и/или knowledge-based). Обычно выражаются через свободную энергию Гиббса ΔG для оценки аффинности (Ki):
   `ΔG = ΔH − TΔS`
   - ΔG < −8 ккал/моль — хорошее связывание; связь с Ki: ΔG = RT·ln(Ki)
   - Применение: MM-PBSA/MM-GBSA (постобработка МД)

**Проблема всех скоринговых функций:** не учитывают сольватацию и энтропию (S) в полной мере → ложные Hits

## Статья: Shityakov et al. 2014 — cPPT ВИЧ-1 в пенистых вирусных векторах

**Зачем изучали:** Проверить, позволит ли перенос cPPT (центральный полипуриновый тракт, 33 п.н.) из ВИЧ-1 в пенистовирусный (FV) вектор трансдуцировать (доставлять ген) неделящиеся клетки.

**Как делали:** Клетки A549 останавливали афидиколином в фазе G1/S, заражали четырьмя конструктами, эффективность оценивали по eGFP (проточная цитометрия, 48 ч); логистическая доза-ответ модель (MATLAB) верифицировала данные.

**Что получили:** FV-вектор с cPPT ВИЧ-1 дал трансдукцию в 20 раз ниже лентивирусного контроля. Причина — дефектные вирусные частицы: обратная транскриптаза FV не распознаёт чужеродный элемент ВИЧ-1.

**Вывод:** cPPT ВИЧ-1 не функционален в FV-контексте; для неделящихся клеток нужны лентивирусные векторы.

> [!important] Связь с вопросами билета
> Статья показывает: структурно похожий элемент (cPPT) теряет функцию вне «родного» контекста — прямая аналогия фармакофорной специфичности. Логистическая доза-ответ модель в MATLAB иллюстрирует количественное вычислительное предсказание — как скоринговая функция в докинге.
