## Вопрос 1: Алгоритмы белок-белок докинга. Отличия от белок-лиганд докинга

Белок-белковый докинг (**PPI**) предсказывает, как два крупных белка стыкуются. Поиск идёт по всей поверхности — в отличие от белок-лигандного докинга, где маленькая молекула ищет конкретный карман.

Основные алгоритмы:
1. **FFT (быстрое преобразование Фурье)** — ZDOCK, ClusPro. Белки представляются как сетки; FFT за одно вычисление оценивает все положения. Быстро, но без гибкости.
2. **HADDOCK** — использует экспериментальные ограничения (NMR, мутагенез). Стадии: жёсткий докинг → гибкое уточнение → вода. Наиболее точный.
3. **RosettaDock** — Монте-Карло с перебором боковых цепей на интерфейсе.

Ключевые отличия от белок-лигандного докинга: лиганд ~70 атомов vs белок ~тысячи остатков; поиск в кармане 20 Å vs вся поверхность; минуты CPU vs часы на кластере.

## Вопрос 2: Молекулярная динамика — типы, важность, плюсы и минусы

МД — симуляция движения атомов: для каждого атома на каждом шаге (~2 фс) решается F = ma. Детерминированный метод (Нобелевская премия 1999).

**Как именно происходит МД:**
1. **Начальные условия** — берём 3D-структуру (из PDB или докинга): координаты всех атомов белка + лиганда + воды
2. **Силовое поле** задаёт потенциальную энергию системы U(r) — сумма вкладов от связей, углов, торсий, VdW и электростатики
3. **Сила на каждый атом** — производная энергии по координате: F = −dU/dr
4. **Интегрирование уравнения движения** — **алгоритм Верле** (или leap-frog) на каждом шаге (~2 фс):
   ```
   a = F/m           ← ускорение из силы
   v(t+dt) = v(t) + a·dt   ← новая скорость
   r(t+dt) = r(t) + v·dt   ← новое положение
   ```
5. **Повторить** для следующего шага → 50–500 нс = миллионы шагов
6. **Траектория** — набор «снимков» системы каждые 1–10 пс → из неё считают RMSD, RMSF, MM-PBSA

**Зачем делается МД после докинга?**
Докинг — статичная «фотография»: даёт позу лиганда в кармане, но не учитывает гибкость белка, растворитель и энтропию. МД — «видео»: проверяет, остаётся ли комплекс стабильным во времени, и даёт более точную ΔG. Типичная связка:
1. Докинг → лучшие позы кандидатов
2. МД (100–500 нс) → проверка стабильности: RMSD лиганда, водородные связи
3. MM-PBSA/MM-GBSA → количественная ΔG связывания (точнее скоринга, медленнее)

Таким образом, МД — **валидация и уточнение** результатов докинга, а не замена ему.

| Тип | Суть | Масштаб | Когда применять |
|---|---|---|---|
| **Классическая (ММ)** | Силовые поля (AMBER, CHARMM), атомы = шары | фс–мкс | Белок-лигандные комплексы, мембраны |
| **Ab initio / QM** | Электроны из первых принципов (DFT) | пс, ≤ сотен атомов | Химические реакции, разрыв связей |
| **QM/MM** | Активный сайт — квантово, остальное — классически | нс, тысячи атомов | Ферментативные реакции с лигандом |
| **Coarse-grained (КЗ)** | Группы атомов → одна «бусина» | мкс–мс, миллионы атомов | Сворачивание белка, сборка мембран |

Плюсы: детализация динамики, количественный расчёт ΔG, учёт гибкости белка.
Минусы: требует GPU/кластер; «застревание» в локальных минимумах; классическая МД не описывает разрыв связей.

## Вопрос 3: Химические расчёты в рациональном дизайне лекарств

Три уровня:
1. **Квантовая химия (DFT)** — карты электростатического потенциала, 3D-PSA (полярная поверхность с учётом конформации). Только для малых молекул.
2. **Докинг и МД** — аффинность связывания (ккал/моль) и динамическая стабильность комплекса.
3. **ADME-фильтры** (всасывание, распределение, метаболизм, выведение):
   - Правило Липинского: MW < 500 Да, logP < 5, HBD ≤ 5, HBA ≤ 10 — пероральная биодоступность.
   - logBB (проницаемость ГЭБ — гематоэнцефалического барьера): logBB = 0.152×ClogP − 0.0148×TPSA + 0.139; порог ≥ 0.3.
   - TPSA (топологическая полярная поверхность) — дескриптор, коррелирующий с всасыванием.

QSAR (количественное соотношение структура–активность) — модель, связывающая дескрипторы с активностью. Предсказывает IC50 или Ki для несинтезированных молекул.

## Статья: Kovalenko et al. 2023 — клик-химия для ингибиторов D3R

**Зачем:** Найти ингибиторы рецептора дофамина D3 (мишень при шизофрении), проникающие через ГЭБ.

**Как:** In silico клик-реакция в RDKit — из 143 075 азидов получили 18 838 триазолов → фильтр Липинского → logBB ≥ 0.3 → докинг AutoDock Vina + iDock (корреляция r = 0.96) → 200 нс МД в AMBER с мембраной POPC:POPE:холестерин.

**Результат:** Хит ID 7000: аффинность −10.16 ккал/моль (референс этиклоприд −7.64); logBB = 0.34 (этиклоприд −0.35). МД: ΔG_lie = −60.17 vs −58.37 ккал/моль — превосходство за счёт Ван-дер-Ваальса (ΔE_vdW < −50 ккал/моль).

**Вывод:** Клик-реакция + logBB-фильтрация — рабочий подход для разработки ЦНС-активных антипсихотиков.
