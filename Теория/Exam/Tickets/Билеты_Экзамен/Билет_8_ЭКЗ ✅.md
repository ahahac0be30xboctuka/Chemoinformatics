## Вопрос 1: ПО для молекулярной динамики и примеры применения

МД — метод, где атомы движутся по законам Ньютона (F = ma), а компьютер отслеживает каждый атом во времени.

| ПО                        | GPU      | Лучшее для                       | Силовые поля                       | Особенности                                           |
| ------------------------- | -------- | -------------------------------- | ---------------------------------- | ----------------------------------------------------- |
| **GROMACS**               | Да       | Белки, мембраны, ДНК             | AMBER, CHARMM, GROMOS, OPLS        | Быстрый, популярен в академии; CPPTRAJ для анализа    |
| **AMBER**                 | Да       | Белок-лигандные комплексы        | AMBER ff14SB/ff19SB, GAFF, Lipid21 | MM-PBSA прямо в пакете; лучший выбор для фармы        |
| **NAMD**                  | Да       | Суперкомпьютеры, нанобиотех      | CHARMM, AMBER                      | Визуализация через VMD; масштабируется на тысячи ядер |
| **CHARMM**                | Частично | Нуклеиновые кислоты, методология | CHARMM36                           | Широкий набор аналитических инструментов              |
| **Desmond** (Schrödinger) | Да       | Фармацевтика, HTS                | OPLS4                              | 60–80× быстрее CPU; интегрирован с Maestro            |
| **OpenMM**                | Да       | ML-потенциалы, Python-интеграция | AMBER, CHARMM, OpenFF              | Гибкость: плагин для нейросетевых силовых полей       |

## Вопрос 2: Силовые поля в МД: типы, плюсы и минусы

**Силовое поле** — набор уравнений и параметров, описывающих взаимодействия между атомами: длины связей, углы, торсионы (вращение вокруг связей), электростатику (закон Кулона) и Ван-дер-Ваальс (потенциал Леннарда-Джонса). Основные семейства:

- **AMBER** (ff14SB, ff19SB) — для белков; GAFF — для органических лигандов; Lipid21 — для мембран.
- **CHARMM36** — для нуклеиновых кислот и мембран.
- **GROMOS** — открытый код, используется в GROMACS.
- **OPLS-AA** — для фармацевтических молекул, используется в Desmond.

Модели воды: TIP3P (быстрая, стандартная), OPC (точная, с ff19SB).

**Плюсы:** быстрые вычисления, системы до миллиона атомов, хорошо валидированы. **Минусы:** не описывают разрыв/образование связей, могут давать артефакты, результаты разных полей трудно сравнивать.

## Вопрос 3: Ключевые параметры МД для оценки белка

- **RMSD** (среднеквадратичное отклонение от начальной структуры) — стабильность системы.

  Формула: `RMSD = √(1/N · Σ(rᵢ − r₀ᵢ)²)` — N атомов, rᵢ — текущие координаты, r₀ᵢ — начальные.

  - Вышел на плато (~1–3 Å) → система равновесна, симуляция корректна
  - Резкий скачок → конформационный переход или нестабильность комплекса
  - RMSD > 5 Å → белок сильно изменился, возможно лиганд покинул карман

  Что считают отдельно: RMSD по всему белку (общая стабильность), по активному сайту, по лиганду (не ушёл ли из позы после докинга).

  **Связь с докингом:** если после МД RMSD лиганда < 2 Å относительно докинговой позы → докинг был правильным.
  
- **RMSF** (флуктуации по остаткам) — гибкость каждого аминокислотного остатка. Коррелирует с B-фактором рентгеновской кристаллографии.
- **Rg** (радиус вращения) — компактность. Растёт при разворачивании белка.
- **Водородные связи** — стабильность белок-лигандного комплекса: больше связей → стабильнее.
- **MM-PBSA / MM-GBSA** — свободная энергия связывания (ΔG). Точнее докинга, но медленнее.

## Статья: Ergodicity Breaking and Self-Destruction of Cancer Cells

**Зачем изучали:** Проверить, может ли усиленный геномный хаос (хромотрипсис — катастрофическая перестройка хромосом) приводить к самоуничтожению раковых клеток.

**Как делали:** Монте-Карло симуляция мутаций гена BCL2 (белок, подавляющий апоптоз — гибель клетки) на Python; стабильность мутанта — через Rosetta; докинг белок-белок — ZDOCK; эксперимент на рабдомиосаркоме (опухоль мышц) крыс.

**Что получили:** При ~10% клеток с хромотрипсисом вероятность мутации BCL2 → 1. Мутант G5W дестабилизирован: ΔES = +4,11 REU, энергия связывания −11,1 ккал/моль против −11,3 у нормального белка → апоптоз запускается.

**Вывод:** Индукция геномного хаоса — потенциальная противораковая стратегия, но есть риск лекарственной устойчивости.

> [!important] Связь с вопросами билета
> Статья применяет инструменты из вопросов билета: RMSD для сравнения белковых структур, докинг (ZDOCK) для оценки взаимодействий, Монте-Карло — аналог методов конформационного поиска в МД.
