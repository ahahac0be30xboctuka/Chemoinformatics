# NeuroClick: software for mimicking click reaction to generate drug-like molecules permeating the blood–brain barrier

> [!abstract] Краткое резюме
> Авторы разработали **NeuroClick** — GUI-программу (Python 3 / RDKit / PyQt5) для in silico моделирования азид-алкинового циклоприсоединения (click-реакции) с целью генерации комбинаторных библиотек 1,2,3-триазолов. Программа фильтрует результаты по правилу Липинского и коэффициенту проникновения через гематоэнцефалический барьер (logBB). Скорость генерации — ~9 000 молекул/мин; тестирование на 143 075 азидах + 1 алкин дало 284 770 триазолов, из которых 1 056 прошли фильтр Clark logBB ≥ 0.3.

---

## 🎯 Цель и гипотеза

**Проблема:** Существующие библиотеки для виртуального скрининга либо ограничивают доступное химическое пространство (уже синтезированные соединения), либо генеративные ML-модели предлагают синтетически нереалистичные структуры.

**Гипотеза:** Перенос реальных органических реакций (click chemistry) в виртуальную среду позволит генерировать химически обоснованные, синтетически доступные библиотеки, пригодные для виртуального скрининга.

**Цель:** Разработать Python-программу, выполняющую in silico азид-алкиновое циклоприсоединение с последующей фильтрацией по drug-likeness и BBB-проникновению.

## 🧪 Методология

### Click-реакция: азид–алкиновое циклоприсоединение
- **Реакция**: азид (R–N₃) + алкин (R'–C≡C–R'') → 1,2,3-триазол
- **Региоселективность:**
  - Тепловая реакция → смесь 1,4- и 1,5-изомеров
  - Cu-катализ (CuAAC) → преимущественно **1,4-изомер**
  - Ru-катализ (RuAAC) → преимущественно **1,5-изомер**
- В NeuroClick реализована **интермолекулярная** реакция (не интрамолекулярная)

### Реализация алгоритма
- Язык: **Python 3**
- Химическая обработка: **RDKit 2022.3.1**
- GUI: **PyQt5 v5.15.6**
- Формат данных: **SMILES** (более лёгкий и быстрый, чем PDB)
- SMARTS-паттерны для задания реакции:
  - `[*:6][N:1]=[N:2]=[N:3].[#6:7][C:4]#[C:5]` → 1,4-изомер
  - `[*:6][N:1]=[N:2]=[N:3].[#6:7][C:4]#[C:5]` → 1,5-изомер

### Фильтрация продуктов

**Правило Липинского (Rule of Five):**
- Молекулярная масса < 500 Да
- ClogP ≤ 5
- Доноры водородных связей ≤ 5
- Акцепторы водородных связей ≤ 10

**Фильтр по logBB** (проникновение через ГЭБ):
$$\text{logBB} = \lg\frac{C_{\text{brain}}}{C_{\text{plasma}}}$$

- **Модель Clark:** logBB = 0.152·ClogP − 0.0148·TPSA + 0.139
- **Модель Rishton:** logBB = 0.155·ClogP − 0.01·TPSA + 0.164
- Порог по умолчанию: logBB ≥ **0.3**

## 📊 Ключевые результаты

### Тестирование алгоритма
- Малый набор: 7 алкинов + 6 азидов → 23 корректных продукта ✅
- Воспроизведение синтеза лигандов дофаминового D3R (по Keck et al., 2015): 10 + 12 молекул ✅

### Масштабный эксперимент: 143 075 азидов + 1 алкин (1-(hex-5-yn-1-yl)-4-phenylpiperazine)

| Этап | Результат |
|------|-----------|
| Входной набор азидов | 143 075 |
| После фильтрации (SMILES-валидация) | 133 632 (93%) |
| Скорость загрузки | ~30 000 молекул/мин |
| Сгенерировано триазолов (1,4- + 1,5-изомеры) | **284 770** |
| Скорость генерации | ~9 000 молекул/мин |
| Время генерации | ~17 мин |
| Прошли фильтр Липинского | 75 190 (26%) |
| Прошли logBB Clark ≥ 0.3 | **1 056** (0.37% от всех) |
| Прошли logBB Rishton ≥ 0.3 | **9 132** (3.2% от всех) |

### P-гликопротеин (Pgp) анализ (ADMETlab 2.0)
- Молекулы после фильтра Clark:
  - 95% — низкая вероятность быть субстратом Pgp (< 0.1)
  - 4% — умеренная вероятность (0.1–0.5)
  - 1% — вероятные субстраты Pgp (> 0.5)

## 💡 Выводы авторов

1. **NeuroClick** — первый современный инструмент для in silico click-реакции с интегрированной фильтрацией по drug-likeness и BBB-проникаемости
2. Скорость ~9 000 молекул/мин позволяет генерировать миллионные библиотеки
3. Использование SMILES вместо PDB кратно ускоряет вычисления
4. Комбинация Lipinski + logBB-фильтра позволяет сократить пространство поиска до синтетически релевантных CNS-активных кандидатов
5. Программа доступна в **GitHub**: https://github.com/Anaiya798/NeuroClick (Windows, Linux, macOS)

**Планируемые улучшения:** другие типы click-реакций (интрамолекулярная, обратная Diels-Alder), дополнительные параметры (logD, Pgp efflux).

## ❓ Возможные вопросы на экзамене

1. **Что такое click-реакция и почему её переносят в in silico среду?**
   → Азид-алкиновое циклоприсоединение, быстрая, селективная, высокий выход; in silico версия позволяет перебрать миллионы комбинаций без синтеза

2. **Чем отличаются 1,4- и 1,5-изомеры триазола? Как управлять региоселективностью?**
   → Cu-катализ → 1,4; Ru-катализ → 1,5; температурная реакция → смесь

3. **Как в NeuroClick реализован фильтр BBB? Какие уравнения используются?**
   → Модели Clark и Rishton (функция ClogP и TPSA); порог logBB ≥ 0.3

4. **Каков главный недостаток медного катализатора в click-химии для биологических применений?**
   → Токсичность меди in vivo; поэтому разрабатываются безмедные (биоортогональные) реакции

5. **Как соотносится NeuroClick со статьёй Kovalenko 2023 (Билет 7)?**
   → NeuroClick — это программная реализация алгоритма, впервые описанного в Kovalenko 2023
