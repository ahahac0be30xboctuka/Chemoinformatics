# Using novel click chemistry algorithm to design D3R inhibitors as blood–brain barrier permeants

> [!abstract] Краткое резюме
> Статья описывает новый in silico подход на основе алгоритма клик-реакции для виртуального синтеза и скрининга ингибиторов рецептора дофамина D3 (D3R) с учётом BBB-проницаемости. Из библиотеки 18 838 триазольных соединений был идентифицирован хит-кандидат (ID 7000) — 1-{4-[1-(decahydronaphthalen-1-yl)-1H-1,2,3-triazol-5-yl]butyl}-4-phenylpiperazine — с превосходящим сродством к D3R и лучшей BBB-проницаемостью по сравнению с референсным лигандом этиклопридом. Результаты подтверждены 200-нс молекулярной динамикой.

---

## 🎯 Цель и гипотеза

**Цель:** Разработать новый workflow виртуального скрининга лигандов D3R с одновременной оптимизацией BBB-проницаемости с использованием in silico клик-реакции.

**Проблема:** D3R — перспективная мишень для лечения шизофрении, болезни Паркинсона, наркотической зависимости, депрессии. Главные препятствия:
1. Высокое структурное сходство D2R и D3R (78% идентичность последовательности, включая сайты связывания) — трудность обеспечения селективности.
2. Необходимость BBB-проницаемости для действия в ЦНС.

**Гипотеза:** In silico клик-реакция (1,3-диполярное циклоприсоединение азида к алкину) позволяет генерировать комбинаторные библиотеки 1,2,3-триазольных соединений. Фильтрация по logBB даст BBB-проницаемые хиты с высоким сродством к D3R.

**Шаблон-скаффолд:** 1-phenyl-4-[4-(1H-1,2,3-triazol-5-yl)butyl]piperazine (с 1,2,3-триазолом вместо амидной группы — биоизостерная замена для повышения метаболической стабильности). Референсный лиганд: **этиклоприд** (эталонный антагонист D3R).

## 🧪 Методология

### 1. Генерация библиотеки (in silico клик-реакция)

- PubChem поиск: 143 075 молекул-азидов (в виде SMILES)
- Фильтрация SMILES: 9 776 уникальных азидов
- **RDKit** (Python, v.2020.09.1.0) — генерация продуктов клик-реакции по SMARTS-паттерну
- Генерируются оба изомера: **1,4-** и **1,5-** (CuAAC и RuAAC соответственно)
- Фильтрация по правилу Липинского: MW < 500, logP ≤ 5, HBD ≤ 5, HBA ≤ 10
- Итоговая библиотека: **18 838** триазольных соединений (9 419 1,4-изомеров + 9 419 1,5-изомеров)

### 2. Фильтрация по BBB-проницаемости (logBB)

Использованы два уравнения для расчёта logBB:
- **Clark:** logBB = 0.152 × ClogP − 0.0148 × TPSA + 0.139
- **Rishton:** logBB = 0.155 × ClogP − 0.01 × TPSA + 0.164

Дескрипторы рассчитаны в RDKit: ClogP и TPSA.

Порог BBB-проницаемости: **logBB ≥ 0.3** (предложен Shityakov et al. и Kunwittaya et al.)

- Прошли Clark: **230 молекул**
- Прошли Rishton: **3 062 молекулы** (все 230 от Clark включены)
- logBB этиклоприда: −0.35 (Clark), −0.02 (Rishton) → ниже порога → сравниваемые хиты должны быть лучше референса

### 3. Молекулярный докинг

- Структура рецептора: **кристаллическая структура D3R** с этиклопридом (PDB: 3PBL)
- Подготовка рецептора: **AutoDockTools4** (жёсткий рецептор)
- Генерация 3D-конформаций лигандов: **Open Babel v.3.1.0** (MMFF94, slow, 3 конформации/молекула)
- Два алгоритма докинга:
  - **AutoDock Vina v.1.1.2** — основной
  - **iDock v.2.2.3** — параллельный для верификации
- Grid box: 20 Å, центр — сайт связывания этиклоприда
- Сервер: Linux, 20 CPU на докинговый запуск

**Референсные аффинности:**
| Соединение | Vina (ккал/моль) | iDock (ккал/моль) |
|---|---|---|
| Скаффолд | −7.74 ± 0.04 | −7.74 ± 0.04 |
| Этиклоприд | −7.64 ± 0.38 | −7.98 ± 0.10 |

### 4. Молекулярная динамика (200 нс)

- ПО: **AMBER 20** (GPU)
- Силовые поля: lipid14 (мембрана), ff99SB (белок), GAFF (лиганд)
- Мембрана: **POPC:POPE:холестерин (100:100:50)** — двойной слой (CHARMM-GUI)
- Анализ: **Linear Interaction Energy (LIE)** — ΔG_lie на основе Ван-дер-Ваальса (ΔE_vdW) и электростатических (ΔE_elec) взаимодействий

## 📊 Ключевые результаты

### Библиотека
- 73% сгенерированных соединений имеют 5–10 вращающихся связей (характерно для лекарственных молекул)
- Корреляция Vina vs iDock: **r = 0.96** (Pearson) — очень высокое согласие между двумя методами
- 1,4- vs 1,5-изомеры: корреляция аффинностей r = 0.73; нет статистически значимого предпочтения изомера

### Топ-10 хитов (оба алгоритма + logBB > 0):
Все топ-10 молекул имеют положительный logBB. Большинство — 1,5-изомеры.

Лучший хит — **соединение ID 7000**:
- Структура: 1-{4-[1-(decahydronaphthalen-1-yl)-1H-1,2,3-triazol-5-yl]butyl}-4-phenylpiperazine
- Vina: **−10.16 ккал/моль** (vs −7.64 у этиклоприда)
- iDock: **−10.17 ккал/моль**
- logBB Clark: **0.34** (выше порога 0.3)
- logBB Rishton: **0.56**
- Уникальность: decahydronaphthalen-1-yl группа принудительно занимает весь карман связывания D3R

### Молекулярная динамика (200 нс):
| Параметр | ID 7000 (новый хит) | Этиклоприд (референс) |
|---|---|---|
| ΔG_lie (ккал/моль) | **−60.17** | −58.37 |
| ΔE_vdW | **< −50 ккал/моль** (ниже порога) | > −50 ккал/моль |
| ΔE_elec | ~−100 ккал/моль | ~−100 ккал/моль |

- Превосходство хита обусловлено прежде всего **более высоким вкладом Ван-дер-Ваальса** (ΔE_vdW < −50 ккал/моль)
- Электростатический вклад схожий для обоих лигандов (катионная природа обоих)

## 💡 Выводы авторов

1. Новый алгоритм комбинирует **in silico клик-реакцию + фармакокинетическую фильтрацию (logBB)** для виртуального скрининга лигандов D3R.
2. Фильтрация по уравнению Ришона предпочтительнее Clark: даёт более широкий набор BBB-проницаемых молекул без потери доли высокоаффинных.
3. Соединение **ID 7000 (1,5-изомер)** превосходит этиклоприд как по аффинности к D3R, так и по BBB-проницаемости.
4. Превосходство хита подтверждено 200-нс МД: ΔG_lie = −60.17 vs −58.37 ккал/моль.
5. Workflow применим для создания новых антипсихотических соединений.
6. Следующий шаг: экспериментальное тестирование in vitro/in vivo.

## ❓ Возможные вопросы на экзамене

1. **Что такое in silico клик-реакция и как она реализована в RDKit?** (SMARTS-паттерн для 1,3-диполярного циклоприсоединения, генерация 1,4- и 1,5-изомеров)
2. **Какой порог logBB используется для фильтрации BBB-проницаемых молекул и чем обоснован?** (logBB ≥ 0.3; Shityakov et al., Kunwittaya et al.)
3. **Почему ID 7000 лучше этиклоприда по результатам МД?** (ΔG_lie −60.17 vs −58.37; за счёт ΔE_vdW < −50 ккал/моль)
4. **Зачем использовали два алгоритма докинга (Vina и iDock)?** (Верификация, r=0.96 подтверждает надёжность результатов)
5. **Почему 1,2,3-триазол используется вместо амидной группы?** (Биоизостерная замена: амид — потенциальный сайт метаболизма; триазол повышает метаболическую стабильность)
6. **Как учитывалась мембрана в МД?** (POPC:POPE:холестерин 100:100:50, CHARMM-GUI, AMBER 20, lipid14)
