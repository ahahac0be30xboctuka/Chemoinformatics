# Lead expansion and virtual screening of Indinavir derivate HIV-1 protease inhibitors using pharmacophoric-shape similarity scoring function

> [!abstract] Краткое резюме
> Авторы применили метод **lead expansion** к известному ингибитору ВИЧ-1 протеазы — индинавиру (IDV) — для генерации 1300 химически разнообразных производных. После многоступенчатого виртуального скрининга (форма → фармакофор → правило Lipinski → докинг → ADME/Tox) был идентифицирован **один новый хит** с иным механизмом связывания с протеазой, не нарушающий правило Lipinski и обладающий улучшенными характеристиками H-связывания по сравнению с IDV.

---

## 🎯 Цель и гипотеза

**Проблема:** ВИЧ-1 протеаза — привлекательная мишень для антиретровирусных препаратов, но известные ингибиторы (в т.ч. индинавир) теряют эффективность из-за множественных мутаций в протеазе вирусных штаммов.

**Гипотеза:** Метод lead expansion позволит создать новый класс ингибиторов протеазы с новым механизмом действия (новым режимом связывания), обходящих мутационную резистентность.

**Цель:** Применить фармакофорно-геометрический скоринг для виртуального скрининга производных индинавира и идентифицировать новый хит с улучшенным профилем связывания.

---

## 🧪 Методология

### 1. Объект исследования
- **Мишень:** ВИЧ-1 протеаза — C2-симметричный аспарагиновый гомодимерный фермент; Asp25 играет ключевую роль в связывании субстрата
- **Исходный лиганд:** Индинавир (Crixivan®, IDV) — мощный ингибитор ВИЧ-1 протеазы

### 2. Подготовка структур
- Кристаллическая структура протеазы взята из PDB (ID: 5362440 — IDV–протеаза комплекс)
- Начальная оптимизация IDV выполнена в программе **MarvinSketch** (ChemAxon)

### 3. Детекция активного сайта
- Программа **PyMol с модулем CAVER** — поиск туннелей и скрытых карманов
- Алгоритм **Дейкстры** для поиска пути от исходного узла
- Определены объём кавитации активного сайта: **282.1 Å³**

### 4. Lead expansion (генерация новых соединений)
- **Генетический алгоритм** → 1300 химически разнообразных соединений (70 поколений)

### 5. Многоступенчатый виртуальный скрининг (воронка)
| Этап | Метод | Результат |
|---|---|---|
| Генерация | Генетический алгоритм | 1300 соединений |
| Скрининг по форме | Shape similarity (0.7–0.9) | 500 топ-молекул |
| Фармакофорный фильтр | Фармакофорный скор < 0.8 → исключить | 227 соединений |
| Lipinski Rule of Five | LigandScout | 1 хит (novel hit) |

### 6. Молекулярный докинг
Два программных комплекса:
- **Molegro Virtual Docker** — скоринговая функция MolDock (кусочно-линейный потенциал), re-ranking procedure, разрешение аффинной сетки 0.3 Å
- **AutoDock** — предварительно вычисленные сетки аффинных потенциалов + поисковые алгоритмы

Параметры докинга: внутренняя энергия связывания, H-связи, торсионные углы Sp2–Sp2.

### 7. ADME/Tox анализ
Использованные алгоритмы:
- **Benigni-Rossa rulebase** — канцерогенность и мутагенность (дискриминантный анализ + структурные правила)
- **Verhaar scheme** — токсичность
- **Cramer rules** — классификация токсичности (классы 1–3)
- **START (Structural Alerts for Reactivity in Toxtree)** — биодеградируемость (данные CEPA)

| HBA (≤10) | 7 ✓ | 6 ✓ |
| Нарушений Lipinski | **1** | **0** ✓ |

### Результаты докинга

| Параметр | IDV | Новый хит |
|---|---|---|
| MolDock Score | — | **−185.7** |
| Rerank Score | — | −148.5 |
| Docking Score | — | −191.6 |
| H-bond energy | — | −12.8 |
| Energy Minimum (ккал/моль) | −15.1 | **−13.8** |
| Число H-связей | 9 | **10** |

> [!warning] Важно
> Минимальная энергия нового хита (−13.8 ккал/моль) **выше** (т.е. менее отрицательная), чем у IDV (−15.1 ккал/моль) → **снижение аффинности**. Это компенсируется новым режимом связывания (дополнительные остатки) и лучшим профилем Lipinski.

### Взаимодействие с протеазой
- **IDV** связывается с: Asp25, Asp29, Gly27
- **Новый хит** принимает тот же режим (Asp25, 29, Gly27) + **дополнительные остатки**: Gly49–Ile50, Asp29–Ala28, Asp30
- Нитрильная группа нового хита создаёт дополнительные H-связи с «лоскутным» регионом («flap region») протеазы
- Длины H-связей: **1.71–2.11 Å** (типичная длина ~2.5 Å → ультракороткие H-связи у IDV: 1, у хита: 2)
- 3D-выравнивание: RMSD = 2.0 Å между «нативной» и «функциональной» конформацией хита

### ADME/Tox результаты
| Параметр | IDV | Новый хит |
|---|---|---|
| Cramer rules | Класс 3 (высокотоксичный) | Класс 3 |
| Биодеградируемость | Класс 2 (нет алертов) | Класс 2 |
| Канцерогенность | Нет | Нет |

> [!note] TPSA и ГЭБ
> TPSA нового хита = 181 Å² — превышает порог для проникновения через гематоэнцефалический барьер (ГЭБ), но оба соединения показывают хорошие транспортные свойства ниже порога 600 Å².

## 💡 Выводы авторов

1. Метод **lead expansion с использованием генетического алгоритма** эффективен для генерации химически разнообразных производных известного лида.
2. **Комбинированный скоринг** (форма + фармакофор + Lipinski + докинг + ADME/Tox) позволяет идентифицировать уникальные соединения с новым механизмом действия.
3. Новый хит: **не нарушает правило Lipinski** (IDV нарушает по MW), образует больше H-связей (10 vs 9), связывается с дополнительными остатками протеазы.
4. Недостаток обоих соединений: гетероциклические кольца со сложными заместителями → **3-й класс токсичности** по Cramer.
5. Исследование демонстрирует применимость виртуального скрининга для **рационального дизайна нового класса ингибиторов ВИЧ-1 протеазы**.

## 🔗 Связь с курсом хемоинформатики

- **Виртуальный скрининг** — центральный метод хемоинформатики; статья — классический пример полного пайплайна
- **Lead expansion** — метод расширения химического пространства вокруг известного хита
- **Генетический алгоритм** — эволюционный оптимизационный алгоритм (тема молекулярного докинга)
- **Фармакофорная модель** — пространственное расположение функциональных групп, важных для активности (тема фармакофоров)
- **Молекулярный докинг** (AutoDock, Molegro) — ключевой метод курса
- **Правило Lipinski (Rule of Five)** — фильтр drug-likeness в ADME
- **TPSA** — дескриптор для предсказания проникновения через мембраны, в т.ч. ГЭБ
- **QSAR** — количественная зависимость структура–активность (ADME/Tox предсказания)
- **CAVER** — алгоритм поиска туннелей в белках (связь с темой активных сайтов)

## ❓ Возможные вопросы на экзамене

1. **Что такое lead expansion и чем он отличается от de novo дизайна?**
   → Lead expansion генерирует производные от известного хита; de novo — строит молекулы с нуля.

2. **Почему IDV нарушает правило Lipinski, а новый хит — нет?**
   → IDV: MW = 613.8 Da (>500 Da). Новый хит: MW = 494.6 Da, все параметры в норме.

3. **Объясните воронку виртуального скрининга в этой статье.**
   → 1300 → 500 (shape) → 227 (фармакофор) → 1 (Lipinski) → докинг + ADME/Tox.

4. **Почему новый хит имеет меньшую аффинность (−13.8 vs −15.1 ккал/моль), но всё же считается перспективным?**
   → Новый режим связывания (дополнительные остатки), больше H-связей, ультракороткие H-связи, лучший профиль Lipinski → потенциально обходит мутационную резистентность.

5. **Какую роль играл алгоритм Дейкстры в этом исследовании?**
   → Для поиска туннелей и кавитаций в структуре протеазы (через CAVER) — идентификация скрытых сайтов связывания.
