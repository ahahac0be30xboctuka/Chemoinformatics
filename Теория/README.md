# Хемоинформатика — Теория

Материалы по курсу хемоинформатики (ИТМО). Конспекты лекций, билеты к экзамену, анализ статей и учебные материалы.

## Структура

```
Теория/
├── Exam/
│   └── Tickets/
│       ├── Билеты_Подробно/      # Развёрнутые ответы на билеты 1–17
│       ├── Билеты_Экзамен/       # Финальные версии билетов для экзамена (+ теорминимум)
│       └── Статьи_анализ/        # Разборы 17 статей экзаменационного списка
│
├── Lections/
│   ├── Links_YT                  # Ссылки на плейлисты лекций на YouTube
│   └── Транскрипции/             # Текстовые конспекты лекций 1–8
│
└── Materials/
    ├── Articles/                 # PDF статей (Article_1–17) + список с библиографией
    └── Books/                    # Учебники по хемоинформатике (PDF)
```

## Материалы

### Лекции
Транскрипции лекций 1–8 в формате Markdown. Видеозаписи: [плейлист 1](https://www.youtube.com/playlist?list=PLVayZSzAgeBWFdS4m_UvAH6euj2kv57v6) · [плейлист 2](https://www.youtube.com/playlist?list=PLVayZSzAgeBUetAYVjP0htk3Vwr5vtYzn)

### Билеты
- **Билеты_Подробно** — полные ответы для подготовки
- **Билеты_Экзамен** — сокращённые версии, готовые к экзамену

### Статьи
17 статей преимущественно из группы Shityakov et al. по темам: BBB-проницаемость, докинг, дескрипторы, SNP-анализ, нанофармакология.
